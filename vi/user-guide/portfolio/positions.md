# Xem vị thế của bạn

## Vị thế hiện tại

Portfolio hiển thị tất cả market bạn đang nắm giữ cổ phần:

```
Vị thế của tôi

Market                              Phía    Cổ phần   Giá TB       Giá trị    L/L
"BTC trên $85K?"                    YES     65        $0.38       $24.70     -$0.38
"Chelsea vs Man City"               NO      100       $0.62       $62.00     +$5.00
"Trump thắng 2028?"                 YES     200       $0.20       $40.00     +$12.00
```

## Ý nghĩa từng cột

| Cột | Ý nghĩa |
|--------|---------|
| **Phía** | YES hoặc NO — bạn đặt cược kết quả nào |
| **Cổ phần** | Số token bạn nắm giữ |
| **Giá TB** | Giá mua trung bình mỗi cổ phần |
| **Giá trị** | Giá trị hiện tại = cổ phần × giá hiện tại |
| **L/L** | Lãi/Lỗ = giá trị hiện tại - chi phí |

## Số dư USDC

USDC khả dụng để giao dịch:

```
Khả dụng: $17,927.54 USDC
Trong lệnh: $150.00 (khóa trong lệnh giới hạn)
Tổng: $18,077.54
```

## Số dư token

Với mỗi market, bạn có thể kiểm tra số dư token YES và NO:

```
Market: "BTC trên $85K?"
  Token YES: 65
  Token NO: 0
  USDC trong lệnh: $0
```
