# Cộng đồng

## Kênh chính thức

| Nền tảng | Liên kết | Mục đích |
| -------- | ---- | ------- |
| **Twitter / X** | [@CoinCrazeCen](https://x.com/CoinCrazeCen) | Tin tức, cập nhật, thông báo |
| **Telegram** | [CoinCrazeCentral](https://t.me/CoinCrazeCentral) | Chat cộng đồng, cập nhật nhanh |
| **GitHub** | [PrediX-Protocol](https://github.com/PrediX-Protocol) | Mã nguồn, issues, đóng góp |

## Cho nhà phát triển

- **GitHub Issues**: Báo lỗi và yêu cầu tính năng
- **Tài liệu**: Bạn đang đọc nó!

## Cho trader

- **Telegram**: Tham gia để nhận thông báo market và thảo luận
- **Twitter/X**: Theo dõi để cập nhật mới nhất

## Đóng góp

PrediX mở cửa cho đóng góp cộng đồng:

1. Fork repository trên GitHub
2. Tạo feature branch
3. Gửi pull request với mô tả rõ ràng
