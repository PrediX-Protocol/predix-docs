# Thuật ngữ

| Thuật ngữ | Định nghĩa |
| ---- | ---------- |
| **AMM** | Automated Market Maker — pool giao dịch thuật toán (Uniswap v4) |
| **CLOB** | Central Limit Order Book — engine khớp lệnh truyền thống |
| **Collateral (Tài sản thế chấp)** | USDC được gửi làm đảm bảo cho token kết quả |
| **Concentrated Liquidity** | Vị thế LP tập trung vào phạm vi giá cụ thể để tăng hiệu quả vốn |
| **Diamond Proxy** | Mẫu nâng cấp EIP-2535 — một proxy, nhiều contract triển khai (facets) |
| **Dynamic Fees (Phí động)** | Phí AMM tăng từ 0.5% đến 5% khi market gần hết hạn |
| **ERC-20** | Tiêu chuẩn token Ethereum — có thể thay thế, tương thích DeFi |
| **ERC-2612** | Mở rộng Permit cho phê duyệt token không gas qua chữ ký |
| **Facet** | Contract triển khai đằng sau Diamond proxy |
| **Hook** | Plugin Uniswap v4 thêm logic tùy chỉnh vào hoạt động pool |
| **Merge (Gộp)** | Đốt 1 YES + 1 NO để nhận 1 USDC |
| **Token NO** | Token ERC-20 trả $1 nếu sự kiện KHÔNG xảy ra |
| **Oracle** | Nguồn dữ liệu bên ngoài báo cáo kết quả sự kiện |
| **Outcome Token (Token kết quả)** | Token YES hoặc NO đại diện cho vị thế trong thị trường dự đoán |
| **Prediction Market (Thị trường dự đoán)** | Nơi người tham gia giao dịch dựa trên kết quả sự kiện tương lai |
| **Redeem (Nhận thưởng)** | Đổi token thắng lấy USDC sau khi market giải quyết |
| **Slippage (Trượt giá)** | Chênh lệch giữa giá giao dịch dự kiến và giá thực tế |
| **Smart Router** | PrediX Router tổng hợp thanh khoản CLOB và AMM |
| **Split (Tách)** | Gửi 1 USDC để nhận 1 YES + 1 NO token |
| **Spread (Chênh lệch)** | Khoảng cách giữa giá bid tốt nhất và giá ask tốt nhất |
| **TVL** | Total Value Locked — tổng USDC được gửi trong hệ thống |
| **USDC** | USD Coin — stablecoin dùng làm tài sản thế chấp (6 decimals) |
| **Token YES** | Token ERC-20 trả $1 nếu sự kiện xảy ra |
