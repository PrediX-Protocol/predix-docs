# Chào mừng đến với PrediX Protocol

PrediX Protocol là nền tảng thị trường dự đoán phi tập trung hoàn toàn, được xây dựng trên **Uniswap v4** và triển khai trên **Unichain L2**.

- **Là gì**: Giao dịch kết quả các sự kiện thực tế bằng token YES/NO — hoàn toàn on-chain, trustless và tương thích với DeFi.
- **Tại sao**: Hệ thống giao dịch kết hợp CLOB + AMM mang lại khớp lệnh tối ưu. Token kết quả ERC-20 mở khóa khả năng tổ hợp DeFi. Diamond proxy cho phép nâng cấp liền mạch.
- **Như thế nào**: Tạo market, giao dịch token kết quả, oracle giải quyết, và nhận thưởng — tất cả chỉ trong vài giao dịch.
