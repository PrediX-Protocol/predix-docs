# 예측 시장

## 예측 시장이란?

예측 시장은 참여자들이 미래 이벤트의 결과에 대해 거래하는 시장입니다. 결과 토큰의 가격은 군중의 집합적 확률 평가를 반영합니다.

## 가격이 확률을 반영하는 방식

각 마켓은 두 개의 토큰을 생성합니다:

- **YES 토큰**: 이벤트가 발생하면 $1.00 지급
- **NO 토큰**: 이벤트가 발생하지 않으면 $1.00 지급

YES가 $0.70에 거래되면, 마켓은 이벤트 발생 확률이 **70%**임을 시사합니다.

## 가격 기준점: YES + NO ≈ $1.00

누구나 할 수 있기 때문에:

- 1 USDC를 1 YES + 1 NO로 **분할**
- 1 YES + 1 NO를 1 USDC로 **병합**

차익거래자들이 이를 유지합니다:

```
YES 가격 + NO 가격 ≈ $1.00

YES + NO > $1.00 → 분할 + 양쪽 매도 → 수익 (가격 하락)
YES + NO < $1.00 → 양쪽 매수 + 병합 → 수익 (가격 상승)
```

이것은 모든 예측 시장의 근본적 불변식입니다.

| Augur | Ethereum의 탈중앙화 AMM | 선구자, 제한적 채택 |

| Omen (Gnosis) | Gnosis Chain의 AMM 기반 | 커뮤니티 주도 |
| **PrediX** | **Uniswap v4의 하이브리드 CLOB+AMM** | **ERC-20 토큰, Diamond 프록시** |
