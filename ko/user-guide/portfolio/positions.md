# 포지션 확인

## 현재 포지션

포트폴리오에 지분을 보유한 모든 마켓이 표시됩니다:

```
내 포지션

마켓                                 사이드   지분      평균가       가치        손익
"BTC $85K 이상?"                    YES     65        $0.38       $24.70     -$0.38
"Chelsea vs Man City"               NO      100       $0.62       $62.00     +$5.00
"Trump 2028 당선?"                  YES     200       $0.20       $40.00     +$12.00
```

## 각 열의 의미

| 열 | 의미 |
|--------|---------|
| **사이드** | YES 또는 NO — 어느 결과에 베팅했는지 |
| **지분** | 보유 토큰 수 |
| **평균가** | 지분당 평균 매수가 |
| **가치** | 현재 가치 = 지분 × 현재 가격 |
| **손익** | 손익 = 현재 가치 - 비용 |

## USDC 잔액

거래 가능한 USDC:

```
사용 가능: $17,927.54 USDC
주문 중: $150.00 (지정가 주문에 잠김)
합계: $18,077.54
```

## 토큰 잔액

각 마켓별 YES, NO 토큰 잔액 확인:

```
마켓: "BTC $85K 이상?"
  YES 토큰: 65
  NO 토큰: 0
  주문 중 USDC: $0
```
