# 지갑 연결하기

## 필요한 것

- Web3 지갑 (MetaMask, Coinbase Wallet, Rainbow 등)
- 가스비를 위한 소량의 ETH (Unichain)

## 단계별 가이드

### 1단계: PrediX 열기

브라우저에서 PrediX 거래 앱을 열어주세요.

### 2단계: "Connect Wallet" 클릭

오른쪽 상단에 있는 **"Connect Wallet"** 버튼을 클릭하세요.

### 3단계: 지갑 선택

목록에서 지갑을 선택하세요:
- **MetaMask** — 가장 인기 있는 브라우저 확장 프로그램
- **Coinbase Wallet** — 초보자에게 쉬움
- **WalletConnect** — 모바일 지갑용
- **Rainbow** — 모바일 우선 지갑

### 4단계: 연결 승인

지갑 팝업이 연결 승인을 요청합니다. **"Connect"** 또는 **"Approve"**를 클릭하세요.

### 5단계: Unichain으로 전환

안내가 나오면, **Unichain Sepolia** 네트워크로 전환하세요:

```
Network Name: Unichain Sepolia
Chain ID: 1301
RPC URL: https://sepolia.unichain.org
Currency: ETH
```

> 💡 대부분의 지갑은 연결 시 네트워크를 자동으로 추가합니다.

### 6단계: 완료!

오른쪽 상단에 지갑 주소가 표시됩니다. 연결 완료!

## 가스비를 위한 ETH가 필요하신가요?

Unichain에서의 트랜잭션은 소량의 ETH가 필요합니다 (거래당 $0.01 미만). Unichain faucet에서 테스트넷 ETH를 받으세요.
