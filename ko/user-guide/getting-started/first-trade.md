# 첫 번째 거래

YES 지분을 사봅시다! 약 2분 걸립니다.

## 1단계: 마켓 선택

마켓 목록을 둘러보세요. 관심 있는 것을 선택하세요. 예:

```
"내일 오후 4시까지 BTC가 $85,000 이상일까?"
YES: $0.38    NO: $0.62
```

마켓을 클릭하여 거래 패널을 엽니다.

## 2단계: YES 또는 NO 선택

- 일어날 것이라 생각하면 **YES** 클릭
- 일어나지 않을 것이라 생각하면 **NO** 클릭

이 예시에서는 **YES**를 매수해 봅시다.

## 3단계: 금액 입력

USDC를 얼마나 쓸지 입력하세요. 소액으로 시작:

```
금액: $25
```

예상치가 표시됩니다:

```
예상 수령: ~65 YES 지분
지분당 가격: ~$0.38
```

## 4단계: "Buy YES" 클릭

초록색 **"Buy YES — $0.38"** 버튼을 클릭하세요.

### 처음이신가요? USDC 승인

첫 거래인 경우, 먼저 USDC를 **승인**해야 합니다:

1. **"Approve USDC"** 클릭
2. 지갑에서 확인
3. 몇 초 대기
4. 이제 **"Buy YES"** 클릭

> 💡 승인은 한 번만 하면 됩니다. 이후 거래에서는 이 단계를 건너뜁니다.

## 5단계: 지갑에서 확인

지갑 팝업이 나타납니다. 세부 사항을 확인하고 **"Confirm"**을 클릭하세요.

## 6단계: 완료!

이제 YES 지분을 소유합니다! 포지션을 확인하세요:

```
내 포지션:
  YES  65 지분
```

## 다음은 무엇?

| YES가 이기면 | NO가 이기면 |
|------------|-----------|
| 65 지분 → $65 USDC | 지분 → $0 |
| 수익: $65 - $25 = **$40** | 손실: **$25** |

**마켓 종료 전 언제든 매도하여** 수익 실현이나 손절이 가능합니다.

## 매도하고 싶으신가요?

1. **"Sell"** 탭으로 전환
2. 매도할 지분 수 입력
3. **"Sell YES"** 클릭
4. USDC가 지갑으로 반환

## 요약

```
1. 마켓 선택 → YES 또는 NO 선택
2. 금액 입력 → $25
3. USDC 승인 (처음만)
4. 매수 → 지갑에서 확인
5. 결과 대기 또는 조기 매도
```

축하합니다 — 이제 예측 시장 트레이더입니다!
