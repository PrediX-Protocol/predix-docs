# 테스트 USDC 받기

PrediX는 모든 거래에 **USDC** ($1 USD에 페깅된 스테이블코인)를 사용합니다. 테스트넷에서는 무료 테스트 USDC를 받을 수 있습니다.

## Faucet — $10,000 테스트 USDC 받기

### 1단계: 연결 확인

지갑이 연결되어 있어야 합니다 ([지갑 연결](connect-wallet.md) 참조).

### 2단계: "Faucet" 또는 "Get USDC" 클릭

PrediX 앱에서 **Faucet** 버튼을 찾아 클릭하세요.

### 3단계: 트랜잭션 확인

지갑이 트랜잭션 확인을 요청합니다. **"Confirm"**을 클릭하세요.

> 💡 소량의 가스비가 필요합니다 (ETH로 $0.01 미만).

### 4단계: USDC 도착

**$10,000 테스트 USDC**가 몇 초 안에 잔액에 표시됩니다. 거래 준비 완료!

## 잔액 확인

USDC 잔액은 거래 패널 상단에 표시됩니다:

```
잔액: $10,000.00
```

## 중요 사항

> ⚠️ **테스트 USDC는 실제 가치가 없습니다.** 이것은 테스트넷입니다 — 학습 및 테스트 전용.

> ⚠️ 필요한 경우 faucet에서 여러 번 USDC를 요청할 수 있습니다.
