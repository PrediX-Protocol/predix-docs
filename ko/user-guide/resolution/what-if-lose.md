# 손실 시 어떻게 되나요?

## 간단한 답변

패배 지분은 $0의 가치가 됩니다. 지분에 지불한 금액만 잃습니다 — 그 이상은 없습니다.

## 예시

```
매수: YES 지분 100개 @ $0.35 → 비용: $35
마켓 해결: NO 승리
YES 지분: $0.00
손실: $35

$35 이상은 절대 잃지 않습니다.
마이너스 잔액은 없습니다.
마진 콜은 없습니다.
```

## 손실을 줄일 수 있나요?

**네 — 마켓 종료 전에 매도하세요.**

```
YES를 $0.35에 매수
가격이 $0.15로 하락 (YES에 불리한 상황)

옵션 A: 보유 → NO 승리 시, 지분당 $0.35 손실
옵션 B: $0.15에 매도 → 지분당 $0.20만 손실 ($0.15 절약)
```

> 💡 **팁**: 예측에 대한 생각이 바뀌면, 조기에 매도하여 손실을 제한하세요.

## 최대 손실

최대 손실은 **항상** 지불한 금액입니다:

| 매수가 | 지분당 최대 손실 | 지분당 최대 수익 |
|-----------|-------------------|-------------------|
| $0.10 | $0.10 | $0.90 |
| $0.50 | $0.50 | $0.50 |
| $0.90 | $0.90 | $0.10 |

## 어떤 조치가 필요한가요?

**아닙니다.** 패배 지분은 단순히 가치가 없습니다. 지갑에 남아있지만 $0의 가치입니다. "종료"하거나 "정산"할 것은 없습니다.
