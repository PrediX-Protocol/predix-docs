# 수익 수령 (Redeem)

## 마켓이 해결되면

마켓이 종료되고 오라클이 결과를 결정한 후:

```
마켓: "BTC가 $85K를 달성할까?" → 결과: YES ✅

YES 지분 → 개당 $1.00의 가치
NO 지분 → $0.00의 가치
```

## 수령 방법

1. 해결된 마켓으로 이동
2. **"Redeem"** 버튼이 표시됩니다
3. **"Redeem"** 클릭
4. 지갑에서 확인
5. USDC가 즉시 지갑으로 전송

```
보유: 65 YES 지분
결과: YES 승리
수령: 65 × $1.00 = $65 USDC → 지갑으로
```

## 졌다면?

**패배** 지분을 보유한 경우:

```
보유: 50 YES 지분
결과: NO 승리
YES 지분 = $0.00
수령할 것 없음. 조치 불필요.
```

> 추가 손실은 없습니다 — 지분 구매에 지불한 금액만 잃습니다.

## YES와 NO 모두 보유한 경우?

```
보유: 30 YES + 20 NO
결과: YES 승리

YES 수령: 30 × $1.00 = $30 USDC ✅
NO 지분: 무가치 ($0) ❌
총 수령: $30
```

## 수령을 잊을 수 있나요?

수령에 **기한이 없습니다**. 해결 후 언제든지 수령할 수 있습니다. 수익은 만료되지 않습니다.

## 다중 결과 마켓

"누가 선거에서 이기나?"와 같은 마켓:

```
보유:
  Trump YES: 50 지분
  Harris YES: 30 지분

승자: Trump (index 0)

수령:
  Trump YES: 50 × $1.00 = $50 ✅
  Harris YES: 무가치 ❌

  또한: Harris NO 지분이 승자!
  (Harris가 이기지 않았으므로)
```
