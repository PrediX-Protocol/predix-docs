# 주문 취소

## 지정가 주문 취소

아직 체결되지 않은 지정가 주문이 있다면:

1. **주문** 섹션에서 미체결 주문 찾기
2. 주문 옆의 **"Cancel"** 버튼 클릭
3. 지갑에서 확인
4. 잠긴 자금이 즉시 반환

```
취소 전: $30 USDC가 BUY 주문에 잠김
취소 후: $30 USDC가 지갑으로 복귀
```

## 무엇이 반환되나요?

| 주문 유형 | 반환되는 것 |
|-----------|----------------|
| BUY YES/NO | USDC (예치금) |
| SELL YES | YES 토큰 |
| SELL NO | NO 토큰 |

## 마켓 만기 후에도 취소할 수 있나요?

**네!** 마켓 만기 후에도 언제든지 주문을 취소할 수 있습니다. 자금이 영원히 잠기지 않습니다.

> 💡 마켓이 만기되었을 때 주문이 아직 열려있다면, 누구나 대신 취소해 줄 수 있습니다 — 하지만 환불은 항상 **주문 소유자**에게 돌아갑니다.

## 부분 체결된 주문

주문이 부분적으로 체결된 경우 (예: 100개 중 40개 체결):

```
원래: BUY 100 YES @ $0.35 → $35 잠김
체결: 40 지분
나머지 취소: 60 지분 → $21 USDC 환불
보유: 40 YES 지분
```
