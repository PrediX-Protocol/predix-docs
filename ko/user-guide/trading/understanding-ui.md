# UI 이해하기

## 메인 레이아웃

거래 페이지는 4개의 주요 영역으로 구성됩니다:

```
┌──────────────────────┬─────────────────┐
│                      │  내 포지션      │
│   가격 차트          │  YES: 65 지분   │
│                      ├─────────────────┤
│                      │  거래 패널      │
│                      │  매수 / 매도    │
│                      │  시장가 / 지정가│
│                      │  [금액]         │
│                      │  [Buy YES]      │
├──────────────────────┤                 │
│   호가창             │  분할 / 병합    │
│   매수    매도       │                 │
│   $0.35   $0.40      │  마켓 정보      │
│   $0.34   $0.41      │                 │
└──────────────────────┴─────────────────┘
```

## 내 포지션 (우측 상단)

현재 이 마켓에서 보유한 것을 표시합니다:

```
내 포지션
  YES    65 지분     ← YES 토큰 65개 보유
  NO     0 지분      ← NO 토큰 미보유
```

## 거래 패널 (우측)

### YES / NO 버튼
```
┌─────────┐ ┌─────────┐
│  YES    │ │   NO    │
│  38¢    │ │   62¢   │
└─────────┘ └─────────┘
```
- 초록색 하이라이트 = 선택됨
- 표시된 가격 = 현재 시장 가격

### 매수 / 매도 토글
- **매수**: USDC를 써서 지분 획득
- **매도**: 지분을 팔아 USDC 획득

### 시장가 / 지정가 토글
- **시장가**: 현재 가격에 즉시 매수/매도
- **지정가**: 원하는 가격 설정, 대기

### 금액 입력
```
금액:                잔액: $17,927.54
┌──────────────────────────────────────┐
│ $ 50.00                         MAX │
└──────────────────────────────────────┘
  [$25]  [$50]  [$75]  [$100]
```
- 금액 입력 또는 프리셋 버튼 클릭
- **MAX** = 전체 잔액
- **잔액** 참고용 표시

### 실행 버튼
```
┌──────────────────────────────────────┐
│         Buy YES — $0.38              │
└──────────────────────────────────────┘
```
수행할 작업과 가격을 표시합니다.

## 가격 차트 (좌측)

시간에 따른 가격 이력을 표시:
- 초록색 선 = 가격 상승
- 빨간색 선 = 가격 하락
- 마우스를 올리면 특정 시점의 가격 표시

## 호가창 (좌측 하단)

```
가격    수량        합계
$0.40    61.58      543.73    ← 매도 호가 (Asks)
$0.39    69.68      482.15
$0.38    70.34      412.47

         $0.38   스프레드: $0.03      ← 중간 가격

$0.35    24.67      24.67     ← 매수 호가 (Bids)
$0.34    31.87      56.54
$0.33    31.29      87.83
```

- **빨간색 (상단)** = 매도 호가 — 이 가격에 매도하려는 사람들
- **초록색 (하단)** = 매수 호가 — 이 가격에 매수하려는 사람들
- **스프레드** = 최고 매수가와 최저 매도가의 차이

## 분할 / 병합 패널

```
┌──────────────────────────────┐
│  [분할]          [병합]      │
│                              │
│  USDC를 동일한 YES + NO     │
│  토큰으로 분할               │
│                              │
│  ┌────────────────────┐      │
│  │ 100.00        USDC │      │
│  └────────────────────┘      │
│                              │
│  ┌────────────────────┐      │
│  │        분할         │      │
│  └────────────────────┘      │
└──────────────────────────────┘
```

- **분할**: $100 USDC → 100 YES + 100 NO 토큰
- **병합**: 100 YES + 100 NO → $100 USDC
- 고급 전략에 유용

## 마켓 정보

```
마켓 정보
  카테고리:          crypto
  총 담보:           $9,936.67 USDC
  상태:              active
  Oracle:            0x699a8c...992cf3
```

- **총 담보**: 이 마켓에 잠긴 USDC 총액
- **상태**: active / ended / resolved
- **Oracle**: 결과를 결정하는 주체
