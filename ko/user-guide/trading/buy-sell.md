# 지분 매수 & 매도

## 지분 매수 (시장가 주문)

시장가 주문은 현재 최적 가격에 즉시 매수합니다.

### YES 매수
1. 마켓 선택 → **YES** 클릭
2. **"Market"** 주문 유형 선택
3. USDC 금액 입력 (예: $50)
4. 예상치 확인: "~130 YES 지분"
5. **"Buy YES — $0.38"** 클릭
6. 지갑에서 확인 → 완료!

### NO 매수
동일한 과정이지만 **NO**를 클릭:
1. **NO** 클릭
2. 금액 입력
3. **"Buy NO — $0.62"** 클릭

## 지분 매도

### YES 매도
1. **"Sell"** 탭으로 전환
2. **YES** 클릭
3. 지분 수 입력 (또는 **MAX** 클릭)
4. 예상치 확인: "~$48 USDC"
5. **"Sell YES"** 클릭

### NO 매도
동일하지만 **NO** 선택.

## 지정가 주문

특정 가격을 원하시나요? **지정가 주문**을 사용하세요.

### 지정가 매수 주문
1. **"Limit"** 주문 유형으로 전환
2. 가격 설정 (예: $0.30)
3. 수량 설정 (예: 100 지분)
4. **"Place Order"** 클릭
5. 누군가가 $0.30에 매도할 때까지 주문 대기

```
비용: 100 × $0.30 = $30 USDC (체결 또는 취소까지 잠김)
```

### 지정가 매도 주문
1. **"Sell"** 탭 → **"Limit"**
2. 가격 설정 (예: $0.70)
3. 수량 설정
4. **"Place Order"** 클릭

### 지정가 주문 취소
- 주문 목록에서 미체결 주문 찾기
- **"Cancel"** 클릭
- 잠긴 USDC 또는 토큰이 즉시 반환

## 지정가 주문 가격 규칙

```
최소 가격: $0.01
최대 가격: $0.99
단계: $0.01 ($0.35는 가능하지만 $0.355는 불가)
최소 주문: $1.00
최대 미체결 주문: 마켓당 50개
```

## 시장가 vs 지정가 주문

| | 시장가 주문 | 지정가 주문 |
|---|---|---|
| 속도 | 즉시 | 원하는 가격까지 대기 |
| 가격 | 현재 최적 가격 | 정확한 지정 가격 |
| 적합한 경우 | 빠른 진입/퇴장 | 더 좋은 가격 확보 |
| 위험 | 슬리피지 가능 | 체결되지 않을 수 있음 |

> 💡 **팁**: 속도가 필요하면 시장가 주문, 가격 관리가 필요하면 지정가 주문을 사용하세요.
