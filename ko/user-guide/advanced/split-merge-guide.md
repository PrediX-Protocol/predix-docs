# 분할 & 병합 (고급)

> 이것은 고급 기능입니다. 대부분의 트레이더는 필요하지 않습니다.

## 분할(Split)이란?

**분할**은 USDC를 동일한 양의 YES와 NO 토큰으로 변환합니다:

```
$100 USDC 분할 → 100 YES 토큰 + 100 NO 토큰
```

## 병합(Merge)이란?

**병합**은 반대 — YES와 NO를 합쳐 USDC로 변환:

```
100 YES 토큰 + 100 NO 토큰 → $100 USDC
```

## 왜 사용하나요?

### 이유 1: 차익거래

YES = $0.40, NO = $0.45 (합계 $0.85)인 경우:

```
$1 분할 → 1 YES + 1 NO (비용: $1)
YES $0.40 매도 + NO $0.45 매도 = $0.85
손실: $0.15 ← 이 방향은 수익 없음

하지만 YES = $0.55, NO = $0.55 (합계 $1.10)인 경우:
YES 1개 ($0.55) + NO 1개 ($0.55) = $1.10 매수
병합 → $1.00
손실: $0.10 ← 이것도 수익 없음

차익거래는 합계 ≠ $1.00일 때만 작동
```

### 이유 2: 양쪽 토큰 확보

마켓 메이커로서 양쪽을 판매하고 싶다면:

```
$500 분할 → 500 YES + 500 NO
SELL YES 주문을 $0.55에 배치
SELL NO 주문을 $0.55에 배치
양쪽에서 수익
```

### 이유 3: 헤지

이미 YES를 보유하고 위험을 줄이고 싶다면:

```
100 YES 보유 ($0.40에 매수)
$50 분할 → 50 YES + 50 NO
현재: 150 YES + 50 NO
NO 승리 시: NO에서 $50 수령 (부분 헤지)
```

## 분할 방법

1. 마켓 페이지로 이동
2. **"Split / Merge"** 섹션 찾기
3. **"Split"** 탭 클릭
4. USDC 금액 입력
5. **"Split"** 클릭
6. 지갑에서 확인

## 병합 방법

1. **"Split / Merge"** 섹션
2. **"Merge"** 탭 클릭
3. 수량 입력 (동일한 YES와 NO 필요)
4. **"Merge"** 클릭
5. USDC가 지갑으로 반환

> ⚠️ 병합하려면 YES와 NO의 **동일한** 수량이 필요합니다. YES 100개, NO 50개를 보유 중이면, 50개만 병합 가능합니다.
