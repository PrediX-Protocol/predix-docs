# 가격은 어떻게 작동하나요?

## 가격 = 트레이더들이 지불하려는 금액

중앙 기관이 가격을 설정하지 않습니다. **트레이더들이 매수·매도를 통해 가격을 결정**합니다.

많은 사람이 YES를 매수 → YES 가격 상승 (더 많은 사람이 발생할 것이라 생각).
많은 사람이 YES를 매도 → YES 가격 하락 (확신이 줄어듦).

## 가격 읽기

```
마켓: "BTC가 연말까지 $100K를 달성할까?"

YES 가격: $0.38 → 시장은 38% 확률로 예상
NO 가격:  $0.62 → 시장은 62% 확률로 일어나지 않을 것으로 예상
```

> 💡 **당신의 우위**: 실제 확률이 시장 가격보다 높거나 낮다고 생각한다면, 거래로 수익을 낼 수 있습니다.

## 수익을 내는 방법

### 시나리오 1: 맞았을 때

```
YES 지분 100개를 $0.38에 매수 → 비용: $38
마켓이 YES로 해결 → 지분당 $1.00
수령: $100
수익: $100 - $38 = $62 ✅
```

### 시나리오 2: 틀렸을 때

```
YES 지분 100개를 $0.38에 매수 → 비용: $38
마켓이 NO로 해결 → 지분당 $0.00
수령: $0
손실: -$38 ❌
```

### 시나리오 3: 조기 매도

```
YES 지분 100개를 $0.38에 매수
가격이 $0.55로 상승 (YES에 유리한 뉴스)
$0.55에 매도 → 수령: $55
수익: $55 - $38 = $17 ✅
(마켓 종료를 기다리지 않고!)
```

## $1 법칙

> ⚠️ **중요**: YES 가격 + NO 가격 ≈ $1.00 (항상)

왜? 1 YES + 1 NO = $1 보장이기 때문입니다. YES + NO < $1이면, 트레이더들이 둘 다 매수하여 차익을 남깁니다. 이것이 가격을 맞추는 원리입니다.

## 가격 변동

| 이벤트 | 영향 |
|-------|--------|
| 결과에 긍정적인 뉴스 | YES 가격 ↑, NO 가격 ↓ |
| 결과에 부정적인 뉴스 | YES 가격 ↓, NO 가격 ↑ |
| 불확실성 증가 | 양쪽 모두 $0.50 방향으로 이동 |
| 결과가 거의 확실 | YES → $0.95+, NO → $0.05 |
