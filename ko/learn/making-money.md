# 수익을 내는 방법

## 전략 1: 매수 후 해결까지 보유

가장 간단한 전략입니다. 저렴하게 매수하고, 결과를 기다립니다.

```
Bitcoin이 $100K를 달성할 것이라 생각 (시장은 35% 확률로만 예상)

→ YES를 $0.35에 매수
→ 마켓 종료를 기다림
→ BTC가 $100K 달성: 지분당 $1.00 → 186% 수익
→ 미달성: 지분당 $0.35 손실
```

**적합한 경우**: 강한 확신이 있는 이벤트.

## 전략 2: 저가 매수, 고가 매도 (스윙 트레이딩)

해결을 기다리지 않습니다. 가격 변동에서 수익을 냅니다.

```
월요일: YES = $0.30 → 100개 매수 ($30)
수요일: 뉴스 발생 → YES가 $0.55로 상승
수요일: 100개 매도 → $55 수령
수익: $25 (2일 만에 83%!)
```

**적합한 경우**: 뉴스를 밀접하게 따르는 활발한 트레이더.

## 전략 3: NO 매수 (반대 베팅)

일어나지 않을 것이라 생각하나요? NO 지분을 매수하세요.

```
마켓: "내일 비가 올까?" → YES = $0.85

날씨 확인: 맑음 예보 → NO를 $0.15에 매수
비가 오지 않으면: NO = $1.00 → 지분당 $0.85 수익 (567%!)
```

**적합한 경우**: 역발상 투자, 과대평가된 이벤트.

## 전략 4: 차익거래 (무위험)

YES + NO < $1.00이면, 위험 없이 수익을 낼 수 있습니다.

```
YES = $0.40, NO = $0.45 → 합계 = $0.85

YES 1개 ($0.40) + NO 1개 ($0.45) = $0.85 매수
둘 중 하나는 반드시 $1.00의 가치
보장 수익: 쌍당 $0.15
```

## 핵심 규칙

### ✅ 해야 할 것

- 거래 전 조사 — 지식이 당신의 무기
- 소액으로 시작 ($25-$50)
- 여러 마켓에 분산 투자
- 예산을 정하고 지키기
- 판단이 바뀌면 매도

### ❌ 하지 말아야 할 것

- 잃을 수 없는 돈 이상으로 투자하지 마세요
- 만기 근처의 마켓을 쫓지 마세요 (수수료가 높음)
- 군중을 맹목적으로 따르지 마세요
- 틀릴 가능성을 무시하지 마세요
- 모든 돈을 하나의 마켓에 넣지 마세요

## 리스크 이해하기

```
최대 손실 = 지분에 지불한 금액
최대 수익 = $1.00 - 지불한 금액

$0.20에 매수 → 최대 손실: $0.20, 최대 수익: $0.80
$0.80에 매수 → 최대 손실: $0.80, 최대 수익: $0.20

낮은 가격 = 높은 잠재 수익 BUT 낮은 승리 확률
```

> ⚠️ **기억하세요**: 예측 시장은 실제 돈이 관련됩니다. 생활에 필요한 돈으로 절대 거래하지 마세요.
