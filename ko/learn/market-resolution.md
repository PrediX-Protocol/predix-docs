# 마켓이 끝나면 어떻게 되나요?

## 타임라인

```
1. 마켓 생성          → 거래 시작
2. 거래 기간          → YES와 NO 지분 매수/매도
3. 마켓 만기          → 거래 중단 (endTime 도달)
4. 해결              → 오라클이 결과 결정 (YES 또는 NO)
5. 수령              → 승자가 승리 지분당 $1.00 수령
```

## 단계별 설명

### 1. 마켓 만기

카운트다운이 0에 도달하면, **더 이상 거래 불가**. 새로운 지분을 사거나 팔 수 없습니다.

### 2. 오라클 해결

**오라클**이 실제 세상의 결과를 결정합니다:

- **가격 마켓** (예: "BTC > $100K?"): Chainlink price feed가 자동으로 확인
- **이벤트 마켓** (예: "누가 선거에서 이기나?"): 관리자가 결과를 확인하고 제출
- 보통 만기 후 수 분에서 수 시간 내에 처리

### 3. 수익 수령

**승리** 지분을 보유하고 있다면:

```
"수령" 클릭 → 승리 지분이 소각 → USDC가 지갑으로 전송

승리 지분 100개 → $100 USDC
```

**패배** 지분을 보유하고 있다면:

```
지분 가치 = $0. 별도 조치 불필요.
단순히 가치가 없습니다.
```

## 조기 퇴장하고 싶다면?

해결을 **기다릴 필요가 없습니다**. 마켓 만기 전 언제든지 지분을 매도할 수 있습니다.

```
YES를 $0.30에 매수 → 현재 가격 $0.60
→ 지금 $0.60에 매도 → 즉시 수익 실현
→ 해결을 기다릴 필요 없음
```

## 오라클이 틀리면?

PrediX에는 안전 메커니즘이 있습니다:

| 메커니즘 | 기능 |
|-----------|-------------|
| **긴급 해결** | 관리자가 7일 후 오버라이드 가능 |
| **환불 모드** | 마켓이 무효인 경우 모두에게 비례 환불 |

> 💡 마켓은 정확하게 해결되도록 설계되었습니다. 긴급 메커니즘은 안전망이며, 거의 사용되지 않습니다.

## 다중 결과 마켓

여러 결과가 있는 마켓 (예: "누가 선거에서 이기나?"):

```
5명의 후보: Trump, DeSantis, Newsom, Harris, Other
→ 관리자가 승자 선택 (예: index 0 = Trump)
→ Trump YES 지분 = $1.00
→ 다른 모든 YES 지분 = $0.00
→ 다른 모든 NO 지분 = $1.00 (해당 후보들이 이기지 않았으므로)
```
