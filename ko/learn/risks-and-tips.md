# 위험과 팁

## 알아야 할 위험

### 1. 돈을 잃을 수 있습니다
지분이 가치 없이 될 수 있습니다. YES를 매수했는데 결과가 NO라면, 지분 = $0.

> ⚠️ **잃어도 감당할 수 있는 금액 이상으로 절대 투자하지 마세요.**

### 2. 시장은 예측 불가능합니다
"확실한 것"조차도 틀릴 수 있습니다. 90% 확률은 10번 중 1번은 예상치 못한 결과가 발생한다는 의미입니다.

### 3. 유동성 부족 위험
일부 마켓은 거래량이 적을 수 있습니다. 이는:
- 원하는 가격에 매수/매도가 어려움
- 스프레드 확대 (매수가와 매도가 차이)
- 더 나쁜 가격을 수용해야 할 수 있음

### 4. 스마트 컨트랙트 위험
PrediX는 스마트 컨트랙트로 운영됩니다. 감사를 받았지만 (264개 테스트, 22개 보안 수정), 어떤 소프트웨어도 100% 무위험은 아닙니다.

### 5. 오라클 위험
마켓 해결은 오라클(데이터 소스)에 의존합니다. 드문 경우, 오라클이 오작동하거나 잘못된 데이터를 제공할 수 있습니다.

## 초보자를 위한 팁

### 소액으로 시작
```
첫째 주: 총 $25-$50
플랫폼 작동 방식을 배우세요
소액으로 실수하세요
```

### 분산 투자
```
1개 마켓에 모든 돈을 넣지 마세요
5-10개 다양한 마켓에 분산
혼합: 암호화폐, 스포츠, 정치
```

### 조사하기
```
무작정 거래하지 마세요
마켓 설명을 주의 깊게 읽으세요
해결 기준을 확인하세요
오라클 출처를 이해하세요
```

### 예산 설정
```
이번 달 거래할 금액을 결정하세요
승패에 관계없이 지키세요
손실을 쫓지 마세요
```

### 지정가 주문 사용
```
시장가 주문은 즉시 체결되지만 슬리피지가 있을 수 있음
지정가 주문으로 정확한 가격을 설정
인내심을 가지세요 — 좋은 가격은 기다리는 자에게 옵니다
```

### 수수료 확인
```
만기 근처에서 거래 수수료 증가:
  일반: 0.5%
  만기 근처: 최대 5%

수수료가 낮을 때 일찍 거래하세요
```

## 흔한 실수

| 실수 | 왜 나쁜가 | 대신 할 것 |
|---------|-------------|-------------------|
| 한 마켓에 올인 | 한 번의 손실 = 전부 잃음 | 여러 마켓에 분산 |
| $0.95에 매수 | 최대 이익 $0.05, $0.95 손실 위험 | 저평가된 마켓 찾기 |
| 해결 기준 무시 | 예상과 다르게 해결될 수 있음 | 설명을 주의 깊게 읽기 |
| 만기 근처에 거래 | 수수료 5%, 스프레드 넓음 | 일찍 거래하거나 지정가 사용 |
| 조기 매도 안 함 | 수익 기회 놓침 | 가격이 유리하게 움직이면 수익 실현 |

## 거래 전 체크리스트

- [ ] 이 마켓이 무엇에 관한 것인지 이해했나요?
- [ ] 해결 기준을 이해했나요?
- [ ] 이 금액을 잃어도 괜찮나요?
- [ ] 현재 가격과 수수료를 확인했나요?
- [ ] 이 마켓에 충분한 유동성이 있나요?
