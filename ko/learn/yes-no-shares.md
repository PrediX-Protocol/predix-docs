# YES와 NO 지분 설명

## 무엇인가요?

PrediX에서 거래할 때, 당신은 **토큰** (디지털 지분)을 구매합니다:

- **YES 토큰** = "이것이 일어날 것이라고 생각합니다"
- **NO 토큰** = "이것이 일어나지 않을 것이라고 생각합니다"

## 가치는 얼마인가요?

**마켓 종료 전** (거래 중):
- YES와 NO는 $0.01에서 $0.99 사이의 시장 가격을 갖습니다
- 가격은 트레이더들의 매수/매도에 따라 변동합니다

**마켓 종료 후** (해결):
- **승리** 토큰 = **$1.00** (USDC로 교환 가능)
- **패배** 토큰 = **$0.00** (가치 없음)

## 예시

```
마켓: "Chelsea가 Man City를 이길까?"

경기 전:
  YES (Chelsea 승리) = $0.35
  NO (Chelsea 미승리) = $0.65

경기 후 — Chelsea 2-1 승리:
  YES = $1.00 ← 승자!
  NO = $0.00

경기 후 — Man City 3-0 승리:
  YES = $0.00
  NO = $1.00 ← 승자!
```

## 지분으로 무엇을 할 수 있나요?

| 행동 | 방법 | 시기 |
|--------|-----|------|
| **매수** | USDC 지불 → YES 또는 NO 수령 | 마켓 종료 전 언제든지 |
| **매도** | YES 또는 NO 매도 → USDC 수령 | 마켓 종료 전 언제든지 |
| **보유** | 해결을 기다림 | 마켓 해결까지 |
| **수령** | 승리 지분 → 개당 $1 USDC | 해결 후 |

## 얼마나 사야 하나요?

지분 수 = 잠재 수령액.

```
YES 50개를 $0.40에 매수 → 비용: $20
YES 승리 시 → 수령: $50 → 수익: $30
NO 승리 시 → 수령: $0 → 손실: $20
```

> 💡 **팁**: 잃어도 괜찮은 만큼만 투자하세요. 배우는 동안 소액($25-$50)으로 시작하세요.

## 다중 결과 마켓

일부 마켓은 2개 이상의 결과를 갖습니다:

```
"2028 미국 대선에서 누가 이길까?"
  → Trump YES/NO 토큰
  → DeSantis YES/NO 토큰
  → Newsom YES/NO 토큰
  → Harris YES/NO 토큰
  → Other YES/NO 토큰

오직 하나의 결과만 이길 수 있습니다. 이길 것이라 생각하는 후보의 YES를 매수하세요.
```
