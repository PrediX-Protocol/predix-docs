# PrediX Protocol에 오신 것을 환영합니다

PrediX Protocol은 **Uniswap v4** 위에 구축되고 **Unichain L2**에 배포된 완전 탈중앙화 예측 시장 플랫폼입니다.

- **무엇**: YES/NO 토큰을 사용하여 실제 세계 이벤트의 결과에 대해 거래하세요 — 완전 온체인, 트러스트리스, DeFi와 조합 가능합니다.
- **왜**: 하이브리드 CLOB + AMM 거래 엔진이 최적의 체결을 제공합니다. ERC-20 결과 토큰은 DeFi 조합성을 실현합니다. Diamond 프록시는 원활한 업그레이드를 가능하게 합니다.
- **어떻게**: 마켓을 생성하고, 결과 토큰을 거래하고, 오라클이 해결하면, 수익을 수령하세요 — 모두 몇 번의 트랜잭션으로 가능합니다.
