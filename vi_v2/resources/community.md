## Nhận hỗ trợ

- **Câu hỏi kỹ thuật:** Kênh `#dev-support` trên Discord
- **Báo bug:** [GitHub Issues](https://github.com/predix-protocol/issues) hoặc [Bug Bounty](../security/bug-bounty.md)
- **Liên hệ hợp tác:** team@predix.io
- **Lỗ hổng bảo mật:** security@predix.io (xem [Bug Bounty](../security/bug-bounty.md))

## Đóng góp

PrediX là mã nguồn mở. Đóng góp luôn được chào đón:

1. Fork repository
2. Tạo feature branch
3. Submit pull request kèm mô tả rõ ràng
4. Tuân theo code style hiện có và thêm test
