# Brand Kit

## Logo

- **Logo chính:** Wordmark PrediX (nền tối)
- **Logo phụ:** Icon PrediX (vuông, dùng cho avatar)
- **Kích thước tối thiểu:** Icon cao 32px, wordmark rộng 120px

## Màu sắc

| Màu | Hex | Dùng cho |
|-------|-----|-------|
| PrediX Navy | `#1B3A5C` | Thương hiệu chính, heading |
| PrediX Blue | `#2E75B6` | Nhấn, link, CTA |
| Success Green | `#22C55E` | Kết quả YES, trạng thái tích cực |
| Error Red | `#EF4444` | Kết quả NO, lỗi |
| Background | `#0F172A` | Nền dark mode |
| Surface | `#1E293B` | Card, panel |
| Text Primary | `#F8FAFC` | Body text (dark mode) |
| Text Secondary | `#94A3B8` | Caption, label |

## Typography

- **Heading:** Inter Bold
- **Body:** Inter Regular
- **Code:** JetBrains Mono

## Nguyên tắc dùng

- Luôn dùng file logo chính thức — không tự vẽ lại hay sửa đổi
- Giữ khoảng clear space tối thiểu bằng 1x chiều cao icon
- Không kéo giãn, xoay, hoặc đổi màu logo
- Nền tối: dùng logo trắng/sáng
- Nền sáng: dùng logo navy/tối
