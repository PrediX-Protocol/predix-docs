# 47. Brand Kit

## Logo

PrediX có hai phiên bản logo chính thức:
•            Logo Full (Wordmark): Chữ “PREDI” trắng + biểu tượng “X” vàng amber với đường kẻ ngang vàng phía dưới. Sử dụng trên nền tối
•            Logo Icon (Symbol): Biểu tượng “X” vàng amber độc lập với đường kẻ ngang. Sử dụng cho favicon, avatar, app icon
Tải logo tại: predixpro.io/brand (SVG, PNG, PDF)

## Color Palette

| Tên | Hex | Sử dụng |
|---|---|---|
| PrediX Gold (Primary) | #F5A623 | Biểu tượng X, accent line, CTA buttons, highlights |
| PrediX Black (Background) | #000000 | Nền chính, header, footer |
| PrediX White (Text) | #FFFFFF | Body text trên nền tối, wordmark “PREDI” |
| Dark Surface | #0A0A0A | Card backgrounds, secondary surfaces |
| Dark Gray | #1A1A1A | Borders, dividers, subtle elements |
| Light Gray | #9CA3AF | Secondary text, captions, placeholders |
| Success Green | #10B981 | YES token, positive outcomes, profit |
| Error Red | #EF4444 | NO token, negative outcomes, loss |

## Typography

- •            Headings: Inter Bold / Semi-Bold
- •            Body: Inter Regular
- •            Code / Addresses: JetBrains Mono
- •        Logo Wordmark: Custom typeface (không thay thế bằng font khác)

## Usage Guidelines

- •            Luôn giữ khoảng cách tối thiểu quanh logo = chiều cao chữ “P” trong wordmark
- •            Không thay đổi tỷ lệ, xoay, làm méo hoặc thêm hiệu ứng (shadow, glow, outline)
- •            Không tách biểu tượng X ra khỏi đường kẻ ngang vàng
- •            Nền tối (khuyến nghị): Logo Full trắng + vàng trên nền đen/tối
- •            Nền sáng: Logo Full đen + vàng trên nền trắng/sáng (có sẵn phiên bản riêng)
- •            Không đặt logo trên nền có pattern phức tạp hoặc hình ảnh nhiều chi tiết
- •            Kích thước tối thiểu: 120px chiều rộng (digital), 30mm (in ấn)
