# Brand Kit

## Logo

- **Primary logo:** PrediX wordmark (dark background)
- **Secondary logo:** PrediX icon (square, for profile images)
- **Minimum size:** 32px height for icon, 120px width for wordmark

## Colors

| Color | Hex | Usage |
|-------|-----|-------|
| PrediX Navy | `#1B3A5C` | Primary brand, headings |
| PrediX Blue | `#2E75B6` | Accent, links, CTAs |
| Success Green | `#22C55E` | YES outcomes, positive states |
| Error Red | `#EF4444` | NO outcomes, errors |
| Background | `#0F172A` | Dark mode background |
| Surface | `#1E293B` | Cards, panels |
| Text Primary | `#F8FAFC` | Body text (dark mode) |
| Text Secondary | `#94A3B8` | Captions, labels |

## Typography

- **Headings:** Inter Bold
- **Body:** Inter Regular
- **Code:** JetBrains Mono

## Usage Guidelines

- Always use the official logo files — do not recreate or modify
- Maintain minimum clear space of 1x the icon height around the logo
- Do not stretch, rotate, or alter logo colors
- On dark backgrounds, use white or light logo variant
- On light backgrounds, use navy or dark logo variant
