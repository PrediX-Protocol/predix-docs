# Community

Connect with the PrediX community and team.

## Official Channels

| Platform | Link | Purpose |
|----------|------|---------|
| **Twitter / X** | [@PrediXProtocol](https://twitter.com/PrediXProtocol) | Announcements, updates |
| **Discord** | [discord.gg/predix](https://discord.gg/predix) | Community chat, support |
| **Telegram** | [t.me/predixprotocol](https://t.me/predixprotocol) | Quick updates, discussion |

## Getting Help

- **Technical questions:** Discord `#dev-support` channel
- **Bug reports:** [GitHub Issues](https://github.com/predix-protocol/issues) or [Bug Bounty](../security/bug-bounty.md)
- **Business inquiries:** team@predix.io
- **Security issues:** security@predix.io (see [Bug Bounty](../security/bug-bounty.md))

## Contributing

PrediX is open-source. Contributions welcome:

1. Fork the repository
2. Create a feature branch
3. Submit a pull request with clear description
4. Follow existing code style and add tests
