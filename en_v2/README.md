# PrediX Protocol

**Decentralized Prediction Markets on Uniswap v4**

PrediX is a fully on-chain prediction market protocol deployed on Unichain (Uniswap's L2). Trade binary outcomes with real ERC-20 tokens, deep hybrid liquidity, and permissionless settlement.

- **What:** Buy and sell YES/NO outcome tokens priced $0.01–$0.99 in USDC. Prices reflect real-time probability.
- **Why:** First prediction market with DeFi-composable ERC-20 tokens, hybrid CLOB+AMM liquidity, and dynamic time-decay fees — all powered by Uniswap v4 Hooks.
- **How:** Smart Router splits your trade across an on-chain order book and AMM pool for best execution. Oracle resolves the outcome. Winners redeem $1 per token.
