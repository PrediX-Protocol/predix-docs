# Community

## Official Channels

| Platform | Link | Purpose |
| -------- | ---- | ------- |
| **Twitter / X** | [@PrediXProtocol] | News, updates, announcements |
| **Discord** | [discord.gg/predix] | Community discussion, support |
| **Telegram** | [t.me/predix] | Quick updates, community chat |
| **GitHub** | [github.com/PrediX-Protocol] | Source code, issues, contributions |
| **Medium / Blog** | [medium.com/@predix] | Long-form articles, research |

## For Developers

- **GitHub Issues**: Report bugs and request features
- **Developer Discord**: #dev channel for technical discussion
- **Documentation**: You're reading it!

## For Traders

- **Trading Discord**: #trading channel for market discussion
- **Twitter**: Follow for new market announcements

## Contributing

PrediX is open to community contributions:

1. Fork the repository on GitHub
2. Create a feature branch
3. Submit a pull request with clear description
4. Join Discord #dev for discussion
