## Understanding Each Column

| Column | Meaning |
|--------|---------|
| **Side** | YES or NO — which outcome you bet on |
| **Shares** | How many tokens you hold |
| **Avg Price** | Average price you paid per share |
| **Value** | Current value = shares × current price |
| **P&L** | Profit/Loss = current value - cost |
