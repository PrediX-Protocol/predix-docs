# Glossary

| Term | Definition |
| ---- | ---------- |
| **AMM** | Automated Market Maker — algorithmic trading pool (Uniswap v4) |
| **CLOB** | Central Limit Order Book — traditional order matching engine |
| **Collateral** | USDC deposited as backing for outcome tokens |
| **Concentrated Liquidity** | LP position focused on a specific price range for capital efficiency |
| **Diamond Proxy** | EIP-2535 upgrade pattern — single proxy, multiple implementation contracts (facets) |
| **Dynamic Fees** | AMM fees that increase from 0.5% to 5% as market approaches expiry |
| **ERC-20** | Ethereum token standard — fungible, composable with DeFi |
| **ERC-2612** | Permit extension for gasless token approvals via signatures |
| **Facet** | An implementation contract behind the Diamond proxy |
| **Hook** | Uniswap v4 plugin that adds custom logic to pool operations |
| **Merge** | Burn 1 YES + 1 NO to receive 1 USDC |
| **NO Token** | ERC-20 token that pays $1 if the event does NOT occur |
| **Oracle** | External data source that reports event outcomes |
| **Outcome Token** | YES or NO token representing a position in a prediction market |
| **Prediction Market** | Marketplace where participants trade on the outcome of future events |
| **Redeem** | Exchange winning tokens for USDC after market resolution |
| **Slippage** | Price difference between expected and executed trade price |
| **Smart Router** | PrediX Router that aggregates CLOB and AMM liquidity |
| **Split** | Deposit 1 USDC to receive 1 YES + 1 NO token |
| **Spread** | Difference between best bid and best ask price |
| **TVL** | Total Value Locked — total USDC deposited in the system |
| **USDC** | USD Coin — stablecoin used as collateral (6 decimals) |
| **YES Token** | ERC-20 token that pays $1 if the event occurs |
