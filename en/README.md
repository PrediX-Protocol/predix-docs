# Welcome to PrediX Protocol

PrediX Protocol is a fully decentralized prediction market platform built on **Uniswap v4** and deployed on **Unichain L2**.

- **What**: Trade on the outcome of real-world events using YES/NO tokens — fully on-chain, trustless, and composable with DeFi.
- **Why**: Hybrid CLOB + AMM trading engine delivers best execution. ERC-20 outcome tokens unlock DeFi composability. Diamond proxy enables seamless upgrades.
- **How**: Create markets, trade outcome tokens, let oracles resolve, and redeem winnings — all in a few transactions.
