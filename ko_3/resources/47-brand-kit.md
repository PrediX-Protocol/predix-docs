PrediX에는 두 가지 공식 로고 버전이 있습니다:

• Logo Full (Wordmark): 흰색 "PREDI" + 앰버 골드 "X" + 아래 골드 가로선. 어두운 배경에 사용

• Logo Icon (Symbol): 앰버 골드 "X" + 가로선만 있는 독립 버전. favicon, 아바타, 앱 아이콘에 사용. 로고 다운로드: predixpro.io/brand (SVG, PNG, PDF)

## Typography

* • Headings: Inter Bold / Semi-Bold
* • Body: Inter Regular
* • Code / Addresses: JetBrains Mono
* • Logo Wordmark: Custom typeface (다른 폰트로 대체 불가)

## Usage Guidelines

* 로고 주변 최소 여백 = 워드마크 "P" 글자 높이
* 비율 변경, 회전, 왜곡, 효과(shadow, glow, outline) 추가 금지
* X 심볼을 가로 골드선과 분리하지 않음
* 어두운 배경 (권장): 검정/어두운 바탕에 흰색 + 골드 Full 로고
* 밝은 배경: 흰색/밝은 바탕에 검정 + 골드 Full 로고 (별도 버전 제공)
* 복잡한 패턴이나 디테일이 많은 이미지 위에 로고 배치 금지
* 최소 크기: 120px 너비(digital), 30mm(인쇄)
