# 내 포지션 확인

## 현재 포지션

포트폴리오는 share를 보유 중인 모든 market을 표시합니다:

```
내 포지션

Market                              방향     Shares    평균가        가치        손익
"BTC $85K 돌파?"                    YES      65        $0.38        $24.70     -$0.38
"Chelsea vs Man City"               NO       100       $0.62        $62.00     +$5.00
"Trump 2028 승리?"                  YES      200       $0.20        $40.00     +$12.00
```

## 각 컬럼의 의미

| 컬럼          | 의미                                      |
| ----------- | --------------------------------------- |
| **방향**      | YES 또는 NO — 베팅한 결과                     |
| **Shares**  | 보유 토큰 수                                 |
| **평균가**     | Share당 평균 매수 가격                        |
| **가치**      | 현재 가치 = Shares × 현재 가격                 |
| **손익**      | 이익/손실 = 현재 가치 − 비용                    |

## USDC 잔액

거래 가능한 USDC:

```
사용 가능: $17,927.54 USDC
주문에 락업: $150.00 (limit order에 락업)
합계: $18,077.54
```

## 토큰 잔액

각 market에서 YES와 NO 토큰 잔액 확인 가능:

```
Market: "BTC $85K 돌파?"
  YES 토큰: 65
  NO 토큰: 0
  주문에 USDC: $0
```
