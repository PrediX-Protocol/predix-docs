# 첫 거래

## 1단계: YES 또는 NO 선택

* 발생할 것이라고 생각하면 **YES** 클릭
* 발생하지 않을 것이라고 생각하면 **NO** 클릭

이 예시에서는 **YES**를 매수합니다.

## 2단계: 금액 입력

사용할 USDC 금액을 입력. 작게 시작:

```
금액: $25
```

예상치를 볼 수 있습니다:

```
예상 수령: ~65 YES shares
주당 가격: ~$0.38
```

## 3단계: "Buy YES" 클릭

녹색 버튼 **"Buy YES — $0.38"**을 클릭.

### 처음이라면? USDC 승인

첫 거래라면 먼저 USDC를 **approve** 해야 합니다:

1. **"Approve USDC"** 클릭
2. 지갑에서 확인
3. 잠시 대기
4. 이제 **"Buy YES"** 클릭

> 💡 Approve는 한 번만 필요. 다음 거래는 이 단계 생략.

## 4단계: 지갑에서 확인

지갑이 팝업을 표시. 세부사항 확인 후 **"Confirm"** 클릭.

## 5단계: 완료!

YES share를 소유하게 되었습니다! 포지션 확인:

```
내 포지션:
  YES  65 shares
```

## 다음은?

| YES 승리                          | NO 승리          |
| ------------------------------- | ------------- |
| 65 shares → $65 USDC            | Shares → $0   |
| 수익: $65 − $25 = **$40**         | 손실: **$25**   |

**Market 종료 전 언제든 매도**해서 이익 실현 또는 손절할 수 있습니다.

## 매도하려면?

1. **"Sell"** 탭으로 전환
2. 매도할 share 수 입력
3. **"Sell YES"** 클릭
4. USDC가 지갑으로 복귀

## 요약

```
1. Market 선택 → YES 또는 NO 선택
2. 금액 입력 → $25
3. USDC 승인 (첫 회만)
4. 매수 → 지갑에서 확인
5. 결과 대기 또는 조기 매도
```

축하합니다 — 이제 예측 시장 트레이더입니다!
