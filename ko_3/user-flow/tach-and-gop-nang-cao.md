# Split & Merge (고급)

> 고급 기능입니다. 대부분의 트레이더는 사용할 필요 없습니다.

## Split이란?

**Split**은 USDC를 동일한 양의 YES와 NO 토큰으로 변환:

```
$100 USDC Split → 100 YES + 100 NO 토큰
```

## Merge란?

**Merge**는 반대 — YES와 NO를 결합해 USDC로 되돌림:

```
100 YES + 100 NO 토큰 → $100 USDC
```

## 왜 사용하나요?

### 이유 1: 가격 차익

YES = $0.40, NO = $0.45 (합 $0.85)인 경우:

```
$1 Split → 1 YES + 1 NO (비용: $1)
YES $0.40 매도 + NO $0.45 매도 = $0.85
손실: $0.15 ← 이 방법으로는 수익 없음

하지만 YES = $0.55, NO = $0.55 (합 $1.10)인 경우:
1 YES 매수 ($0.55) + 1 NO 매수 ($0.55) = $1.10
Merge → $1.00
손실: $0.10 ← 역시 수익 없음

가격 차익은 합계가 $1.00이 아닐 때만 작동
```

### 이유 2: 양쪽 토큰 확보

Market maker로서 양쪽을 매도하고 싶다면:

```
$500 Split → 500 YES + 500 NO
$0.55에 SELL YES 주문
$0.55에 SELL NO 주문
양쪽에서 수익
```

### 이유 3: 헷지

이미 YES를 보유했고 리스크를 줄이고 싶다면:

```
100 YES 보유 ($0.40에 매수)
$50 Split → 50 YES + 50 NO
현재: 150 YES + 50 NO
NO 승리 시: NO에서 $50 수령 (부분 헷지)
```

## Split 방법

1. Market 페이지로 이동
2. **"Split / Merge"** 섹션 찾기
3. **"Split"** 탭 클릭
4. USDC 금액 입력
5. **"Split"** 클릭
6. 지갑에서 확인

## Merge 방법

1. **"Split / Merge"** 섹션
2. **"Merge"** 탭 클릭
3. 수량 입력 (YES와 NO가 동일해야 함)
4. **"Merge"** 클릭
5. USDC가 지갑으로 복귀

> ⚠️ Merge하려면 **동일한 수량의** YES와 NO가 필요. 100 YES와 50 NO만 있다면 50개만 merge 가능.
