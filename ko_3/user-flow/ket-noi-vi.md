# 지갑 연결

### 필요한 것 <a href="#ban-can-gi" id="ban-can-gi"></a>

* Web3 지갑 (MetaMask, Coinbase Wallet, Rainbow 등)
* 가스비용 소량의 ETH (Unichain)

### 단계별 가이드 <a href="#huong-dan-tung-buoc" id="huong-dan-tung-buoc"></a>

#### 1단계: PrediX 열기 <a href="#buoc-1-mo-predix" id="buoc-1-mo-predix"></a>

브라우저에서 PrediX 거래 앱을 엽니다.

#### 2단계: "Connect Wallet" 클릭 <a href="#buoc-2-nhan-connect-wallet" id="buoc-2-nhan-connect-wallet"></a>

우측 상단의 **"Connect Wallet"** 버튼을 클릭합니다.

#### 3단계: 지갑 선택 <a href="#buoc-3-chon-vi" id="buoc-3-chon-vi"></a>

목록에서 지갑을 선택:

* **MetaMask** — 가장 인기 있는 브라우저 확장
* **Coinbase Wallet** — 초보자에게 간편
* **WalletConnect** — 모바일 지갑용
* **Rainbow** — 모바일 우선 지갑

#### 4단계: 연결 승인 <a href="#buoc-4-phe-duyet-ket-noi" id="buoc-4-phe-duyet-ket-noi"></a>

지갑이 연결 요청 팝업을 표시. **"Connect"** 또는 **"Approve"** 클릭.

#### 5단계: Unichain으로 전환 <a href="#buoc-5-chuyen-sang-unichain" id="buoc-5-chuyen-sang-unichain"></a>

요청이 뜨면 **Unichain Sepolia** 네트워크로 전환: Network Name: Unichain Sepolia Chain ID: 1301 RPC URL: https://sepolia.unichain.org Currency: ETH

> 💡 대부분의 지갑은 연결 시 자동으로 네트워크를 추가합니다.

#### 6단계: 완료! <a href="#buoc-6-xong" id="buoc-6-xong"></a>

우측 상단에 지갑 주소가 표시됩니다. 연결 완료!

### 가스비용 ETH가 필요한가요? <a href="#can-eth-cho-phi-gas" id="can-eth-cho-phi-gas"></a>

Unichain 거래는 매우 적은 양의 ETH만 소비합니다 (거래당 $0.01 미만). Unichain faucet에서 testnet ETH를 받을 수 있습니다.<br>
