# 테스트 USDC 받기

PrediX는 모든 거래에 **USDC** ($1 USD에 페그된 스테이블코인)를 사용합니다. Testnet에서는 테스트 USDC를 무료로 받을 수 있습니다.

### Faucet — $10,000 테스트 USDC 수령 <a href="#faucet-nhan-usd10-000-usdc-thu-nghiem" id="faucet-nhan-usd10-000-usdc-thu-nghiem"></a>

#### 1단계: 연결 상태 확인 <a href="#buoc-1-dam-bao-da-ket-noi" id="buoc-1-dam-bao-da-ket-noi"></a>

지갑이 연결되어 있어야 합니다 ([지갑 연결](https://app.gitbook.com/o/wYfbYitvenxfZt7xSZuL/s/bRtr9MNY8QNHV7pD0aua/huong-dan-su-dung/connect-wallet) 참조).

#### 2단계: "Faucet" 또는 "Get USDC" 클릭 <a href="#buoc-2-nhan-faucet-hoac-get-usdc" id="buoc-2-nhan-faucet-hoac-get-usdc"></a>

PrediX 앱에서 **Faucet** 버튼을 찾아 클릭합니다.

#### 3단계: 거래 확인 <a href="#buoc-3-xac-nhan-giao-dich" id="buoc-3-xac-nhan-giao-dich"></a>

지갑이 거래 확인을 요청합니다. **"Confirm"** 클릭.

> 💡 매우 적은 가스비용이 필요합니다 (ETH $0.01 미만).

#### 4단계: USDC 도착 <a href="#buoc-4-usdc-da-den" id="buoc-4-usdc-da-den"></a>

수 초 안에 **$10,000 테스트 USDC**가 잔액에 나타납니다. 거래 준비 완료!

### 잔액 확인 <a href="#kiem-tra-so-du" id="kiem-tra-so-du"></a>

USDC 잔액은 거래 패널 상단에 표시됩니다: 잔액: $10,000.00

### 중요 참고 <a href="#luu-y-quan-trong" id="luu-y-quan-trong"></a>

> ⚠️ **테스트 USDC는 실제 가치가 없습니다.** Testnet입니다 — 학습 및 실험 목적만.

> ⚠️ 필요하면 faucet에서 여러 번 USDC를 요청할 수 있습니다.
