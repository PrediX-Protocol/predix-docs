# 틀리면 어떻게 되나?

## 짧은 답

패배 shares는 $0 가치가 됩니다. 그 shares를 위해 지불한 금액만 잃음 — 그 이상 없음.

## 예시

```
매수: 100 YES shares @ $0.35 → 비용: $35
Market resolve: NO 승리
YES shares: $0.00
손실: $35

$35보다 더 잃지 않음.
마이너스 잔액 없음.
마진 콜 없음.
```

## 손실을 줄일 수 있나요?

**예 — market이 끝나기 전에 매도.**

```
$0.35에 YES 매수
가격이 $0.15로 하락 (YES에 불리한 상황)

옵션 A: 보유 → NO 승리 시 share당 $0.35 손실
옵션 B: $0.15에 매도 → share당 $0.20만 손실 ($0.15 절약)
```

> 💡 **Tip**: 예측에 대해 생각이 바뀌면 조기 매도해서 손실 제한.

## 최대 손실

최대 손실은 **항상** 지불한 금액:

| 매수 가격 | Share당 최대 손실 | Share당 최대 수익 |
| ----- | ---------------- | ---------------- |
| $0.10 | $0.10            | $0.90            |
| $0.50 | $0.50            | $0.50            |
| $0.90 | $0.90            | $0.10            |

## 무엇을 해야 하나요?

**없음.** 패배 shares는 단순히 가치 없음. 지갑에 남지만 $0 가치. "닫기"나 "청산"할 필요 없음.
