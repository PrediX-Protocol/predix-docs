# 주문 취소

## Limit Order 취소

체결되지 않은 limit order가 있는 경우:

1. **주문** 섹션에서 열린 주문 찾기
2. 주문 옆 **"Cancel"** 버튼 클릭
3. 지갑에서 확인
4. 락업된 자금이 즉시 반환

```
취소 전: $30 USDC가 BUY 주문에 락업
취소 후: $30 USDC 지갑으로 복귀
```

## 무엇이 반환되나요?

| 주문 유형      | 반환         |
| ---------- | ---------- |
| BUY YES/NO | USDC (예치금) |
| SELL YES   | YES 토큰     |
| SELL NO    | NO 토큰      |

## Market 만료 후에도 취소 가능한가요?

**예!** Market이 만료된 후에도 주문을 취소할 수 있습니다. 자금은 영구적으로 락업되지 않습니다.

> 💡 Market이 만료되었는데 주문이 열린 상태라면 누구나 대신 취소 가능 — 하지만 자금은 항상 **당신(주문자)**에게 반환됩니다.

## 부분 체결 주문

주문이 부분적으로 체결된 경우 (예: 100 shares 중 40 체결):

```
초기: BUY 100 YES @ $0.35 → $35 락업
체결: 40 shares
나머지 취소: 60 shares → $21 USDC 환불
보유: 40 YES shares
```
