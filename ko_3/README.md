# Welcome

월드컵 결승 예측부터 비트코인 가격 추세까지 — PrediX는 모든 예측을 열린 시장으로 바꿉니다. 사용자는 outcome token을 보유하는 방식으로 자신의 관점을 표현합니다.

PrediX는 Unichain(유니스왑이 운영하는 OP Stack 기반 L2) 위에 구축되었으며, AMM(Uniswap v4 Hooks)과 온체인 오더북(CLOB)을 Smart Router로 결합합니다. 사용자는 USDC로 이진(YES/NO) 시장 또는 다중 결과(multi-outcome) 시장의 outcome token을 매매할 수 있습니다.

더 간단히 말해: 당신이 믿는 결과를 고르고, 그 예측을 대표하는 토큰을 보유하면, 시장은 커뮤니티의 관점을 실시간으로 반영합니다.

**What — PrediX란?**

* 완전 온체인, 비수탁형 prediction market
* **하이브리드 트레이딩 엔진:** AMM + CLOB + 최적 Smart Router
* ERC-20 outcome token (YES/NO) — 전체 DeFi와 호환

**Why — 왜 PrediX인가?**

* 2025년 $64B 시장, 2026년 $200–325B 예상. APAC에 3,800만+ 크립토 사용자가 있지만 현지 prediction market은 없음
* Polymarket은 33개국 이상에서 금지. Kalshi는 미국만 지원. 구조적 공백이 큼

**How — 어떻게 작동하는가?**

* 마켓 생성 → YES/NO 토큰 거래 → Oracle resolve → USDC redeem
* Smart Router가 CLOB + AMM 간 주문을 최적 분배
* Diamond proxy (EIP-2535)로 모듈 업그레이드 가능
