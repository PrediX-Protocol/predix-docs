# 利益を得る方法

## 戦略1：購入して解決まで保有

最もシンプルな戦略。安く買い、結果を待ちます。

```
Bitcoinが$100Kに到達すると予想（マーケットは35%の確率のみ）

→ YESを$0.35で購入
→ マーケット終了を待つ
→ BTCが$100K達成：1枚$1.00 → 186%利益
→ 未達成：1枚$0.35の損失
```

**適している人**: 強い確信を持つイベント。

## 戦略2：安く買って高く売る（スイングトレード）

解決を待ちません。価格変動から利益を得ます。

```
月曜日: YES = $0.30 → 100枚購入 ($30)
水曜日: ニュース発表 → YESが$0.55に上昇
水曜日: 100枚売却 → $55受取
利益: $25 (2日で83%!)
```

**適している人**: ニュースを密にフォローするアクティブトレーダー。

## 戦略3：NOを購入（反対に賭ける）

起こらないと思う？NOシェアを購入しましょう。

```
マーケット: 「明日雨が降る？」 → YES = $0.85

天気予報を確認: 晴れ → NOを$0.15で購入
雨なし: NO = $1.00 → 1枚$0.85の利益 (567%!)
```

**適している人**: 逆張り思考、過大評価されたイベント。

## 戦略4：裁定取引（リスクフリー）

YES + NO < $1.00の場合、リスクゼロで利益を得られます。

```
YES = $0.40, NO = $0.45 → 合計 = $0.85

YES 1枚 ($0.40) + NO 1枚 ($0.45) = $0.85で購入
どちらか一方は必ず$1.00の価値に
確定利益：ペアあたり$0.15
```

## 重要なルール

### ✅ すべきこと

- 取引前にリサーチ — 知識があなたの武器
- 少額から開始 ($25-$50)
- 複数のマーケットに分散
- 予算を決めて守る
- 判断が変わったら売却

### ❌ すべきでないこと

- 失っても困らない額以上を投資しない
- 満期近くのマーケットを追わない（手数料が高い）
- 群衆に盲目的に従わない
- 間違う可能性を無視しない
- 全資金を1つのマーケットに投入しない

## リスクの理解

```
最大損失 = シェアに支払った金額
最大利益 = $1.00 - 支払った金額

$0.20で購入 → 最大損失: $0.20、最大利益: $0.80
$0.80で購入 → 最大損失: $0.80、最大利益: $0.20

低い価格 = 高い潜在利益 だが 低い勝率
```

> ⚠️ **忘れないでください**: 予測市場は実際のお金が関わります。生活に必要なお金では絶対に取引しないでください。
