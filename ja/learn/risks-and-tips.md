# リスクとヒント

## 知っておくべきリスク

### 1. お金を失う可能性があります
シェアが無価値になる可能性があります。YESを購入して結果がNOだった場合、シェア = $0。

> ⚠️ **失っても困らない金額以上は絶対に投資しないでください。**

### 2. マーケットは予測不能です
「確実なこと」でも外れる可能性があります。90%の確率とは、10回に1回は予想外の結果が起こるということです。

### 3. 低流動性リスク
一部のマーケットは取引量が少ない場合があります。これは：
- 希望の価格での売買が困難
- スプレッドの拡大（売買価格の差）
- より悪い価格を受け入れる必要がある場合も

### 4. スマートコントラクトリスク
PrediXはスマートコントラクトで運用されています。監査済み（264テスト、22件のセキュリティ修正）ですが、100%リスクフリーなソフトウェアはありません。

### 5. オラクルリスク
マーケットの解決はオラクル（データソース）に依存します。まれに、オラクルが誤動作したり、誤ったデータを提供する可能性があります。

## 初心者へのヒント

### 少額から始める
```
最初の週: 合計$25-$50
プラットフォームの仕組みを学ぶ
少額で間違えを経験する
```

### 分散する
```
1つのマーケットに全額を入れない
5-10の異なるマーケットに分散
ミックス: 暗号資産、スポーツ、政治
```

### リサーチする
```
盲目的に取引しない
マーケットの説明を注意深く読む
解決基準を確認する
オラクルのソースを理解する
```

### 予算を設定する
```
今月いくら取引するか決める
勝っても負けても守る
損失を追わない
```

### 指値注文を使う
```
成行注文は即座に約定するがスリッページの可能性あり
指値注文で正確な価格を設定
忍耐を持つ — 良い価格は待つ人のもとへ
```

### 手数料を確認する
```
満期近くで取引手数料が増加:
  通常: 0.5%
  満期近く: 最大5%

手数料が低い早い時期に取引する
```

## よくある間違い

| 間違い | なぜ悪いか | 代わりにすること |
|---------|-------------|-------------------|
| 1つのマーケットに全額投入 | 1回の損失 = 全てを失う | マーケットを分散 |
| $0.95で購入 | 最大利益$0.05、$0.95の損失リスク | 割安なマーケットを探す |
| 解決基準を無視 | 予想と異なる解決になる可能性 | 説明を注意深く読む |
| 満期近くに取引 | 手数料5%、スプレッド拡大 | 早めに取引か指値注文を使用 |
| 早期売却をしない | 利益機会を逃す | 価格が有利に動いたら利益確定 |
