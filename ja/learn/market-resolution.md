# マーケット終了後はどうなる？

## タイムライン

```
1. マーケット作成     → 取引開始
2. 取引期間          → YESとNOシェアの売買
3. マーケット満期     → 取引停止（endTime到達）
4. 解決              → オラクルが結果を決定（YESまたはNO）
5. 受取              → 勝者が勝利シェア1枚あたり$1.00を受取
```

## ステップバイステップ

### 1. マーケット満期

カウントダウンがゼロになると、**取引はできなくなります**。新しいシェアの売買はできません。

### 2. オラクルが解決

**オラクル**が現実世界の結果を決定します：

- **価格マーケット**（例：「BTC > $100K？」）：Chainlink price feedが自動で確認
- **イベントマーケット**（例：「選挙の勝者は？」）：管理者が結果を確認・提出
- 通常、満期後数分〜数時間で処理

### 3. 賞金の受取

**勝利**シェアを保有している場合：

```
「受取」クリック → 勝利シェアが焼却 → USDCがウォレットに送金

勝利シェア100枚 → $100 USDC
```

**敗北**シェアを保有している場合：

```
シェアの価値 = $0。特にアクションは不要。
単に価値がなくなります。
```

## 途中で退出したい場合は？

解決を**待つ必要はありません**。マーケット満期前ならいつでもシェアを売却できます。

```
YESを$0.30で購入 → 現在価格$0.60
→ 今$0.60で売却 → 即座に利益確定
→ 解決を待つ必要なし
```

## オラクルが間違えたら？

PrediXには安全メカニズムがあります：

| メカニズム | 機能 |
|-----------|-------------|
| **緊急解決** | 管理者が7日後にオーバーライド可能 |
| **返金モード** | マーケットが無効な場合、全員に比例返金 |

> 💡 マーケットは正確に解決されるよう設計されています。緊急メカニズムは安全ネットであり、めったに使用されません。

## マルチアウトカムマーケット

複数の結果があるマーケット（例：「選挙の勝者は？」）：

```
5人の候補者: Trump, DeSantis, Newsom, Harris, Other
→ 管理者が勝者を選択（例: index 0 = Trump）
→ Trump YESシェア = $1.00
→ 他のすべてのYESシェア = $0.00
→ 他のすべてのNOシェア = $1.00（それらの候補者が勝たなかったため）
```
