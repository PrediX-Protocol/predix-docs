# YESとNOシェアの解説

## 何ですか？

PrediXで取引する際、あなたは**トークン**（デジタルシェア）を購入します：

- **YESトークン** = 「これが起こると思う」
- **NOトークン** = 「これは起こらないと思う」

## 価値はいくら？

**マーケット終了前**（取引中）：
- YESとNOは$0.01から$0.99の間の市場価格を持ちます
- 価格はトレーダーの売買によって変動します

**マーケット終了後**（解決）：
- **勝利**トークン = **$1.00**（USDCと交換可能）
- **敗北**トークン = **$0.00**（無価値）

## 例

```
マーケット: 「ChelseaはMan Cityに勝つ？」

試合前:
  YES (Chelsea勝利) = $0.35
  NO (Chelsea非勝利) = $0.65

試合後 — Chelsea 2-1勝利:
  YES = $1.00 ← 勝者！
  NO = $0.00

試合後 — Man City 3-0勝利:
  YES = $0.00
  NO = $1.00 ← 勝者！
```

## シェアで何ができる？

| アクション | 方法 | タイミング |
|--------|-----|------|
| **購入** | USDC支払い → YESまたはNOを取得 | マーケット終了前いつでも |
| **売却** | YESまたはNOを売却 → USDCを取得 | マーケット終了前いつでも |
| **保有** | 解決を待つ | マーケット解決まで |
| **受取** | 勝利シェア → 1枚$1 USDC | 解決後 |

## いくつ買うべき？

シェア数 = 潜在的な受取額。

```
YES 50枚を$0.40で購入 → コスト: $20
YES勝利時 → 受取: $50 → 利益: $30
NO勝利時 → 受取: $0 → 損失: $20
```

> 💡 **ヒント**: 失っても大丈夫な金額だけ投資してください。学習中は少額（$25-$50）から始めましょう。

## マルチアウトカムマーケット

一部のマーケットは2つ以上の結果を持ちます：

```
「2028年米国大統領選の勝者は？」
  → Trump YES/NOトークン
  → DeSantis YES/NOトークン
  → Newsom YES/NOトークン
  → Harris YES/NOトークン
  → Other YES/NOトークン

1つの結果だけが勝てます。勝つと思う候補のYESを購入しましょう。
```
