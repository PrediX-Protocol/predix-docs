# 予測市場とは？

予測市場は、**現実世界で何が起こるかに賭けて** — 正しければ報酬を得る仕組みです。

## 簡単な例

> **マーケット**: 「Bitcoinは12月31日までに$100,000を超えるか？」

- **YES**だと思う → YESシェアを$0.40で購入
- マーケット終了、BTCは$105,000 → **YES勝利！**
- YESシェアの価値は**1枚$1.00**に
- $0.40で購入、$1.00を受け取り → **利益：1枚あたり$0.60**

BTCが$95,000だった場合？YESシェア = $0。$0.40の損失。

## 従来のベッティングとの違い

| | 従来のベッティング | 予測市場 |
|---|---|---|
| オッズを決めるのは？ | ハウス（ブックメーカー） | マーケット（他のトレーダー） |
| 途中で売れる？ | ❌ 通常不可 | ✅ いつでも可能 |
| 相手は？ | ハウス | 他のトレーダー |
| 透明性 | オッズ非公開 | オンチェーンで価格公開 |
| 資金管理 | 会社が保管 | 自分のウォレット |

## 重要な概念：価格 = 確率

シェアの価格は、マーケットが予想する確率を示します：

| 価格 | マーケットの判断... |
|-------|-----------------|
| $0.10 | 発生確率10% |
| $0.50 | 50% — コイントスレベル |
| $0.80 | 80% — 非常に高い可能性 |
| $0.95 | 95% — ほぼ確実 |

**マーケットと意見が異なれば利益を得られます。** マーケットが30%と言っていても、あなたが70%と考えるなら？$0.30で購入、$1.00を獲得可能。

## 何を取引できる？

- 🏦 **暗号資産**: 「ETHは$10Kを突破する？」
- 🗳️ **政治**: 「次の選挙で誰が勝つ？」
- ⚽ **スポーツ**: 「Chelsea vs Man City — 勝者は？」
- 🌍 **世界情勢**: 「6月までに停戦はある？」
- 📊 **経済**: 「FRBは利下げする？」

## 2つの結果：YESとNO

すべてのマーケットには2つのトークンがあります：

- **YESトークン**：イベントが発生すれば$1、しなければ$0
- **NOトークン**：イベントが発生しなければ$1、すれば$0

YES + NOは常に≈ $1です（どちらか一方が必ず勝つため）。
