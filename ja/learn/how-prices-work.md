# 価格の仕組み

## 価格 = トレーダーが支払う意思のある金額

中央機関が価格を設定するわけではありません。**トレーダーが売買を通じて価格を決定**します。

多くの人がYESを購入 → YES価格上昇（より多くの人が発生すると考えている）。
多くの人がYESを売却 → YES価格下落（確信が薄れている）。

## 価格の読み方

```
マーケット: 「BTCは年末までに$100Kに到達する？」

YES価格: $0.38 → マーケットは38%の確率と予想
NO価格:  $0.62 → マーケットは62%で起こらないと予想
```

> 💡 **あなたの優位性**: 実際の確率がマーケット価格より高い（または低い）と考えるなら、取引で利益を得られます。

## 利益の出し方

### シナリオ1：正しかった場合

```
YESシェア100枚を$0.38で購入 → コスト: $38
マーケットがYESで解決 → 1枚$1.00
受取: $100
利益: $100 - $38 = $62 ✅
```

### シナリオ2：間違っていた場合

```
YESシェア100枚を$0.38で購入 → コスト: $38
マーケットがNOで解決 → 1枚$0.00
受取: $0
損失: -$38 ❌
```

### シナリオ3：途中で売却

```
YESシェア100枚を$0.38で購入
価格が$0.55に上昇（YESに有利なニュース）
$0.55で売却 → 受取: $55
利益: $55 - $38 = $17 ✅
（マーケット終了を待たずに！）
```

## $1ルール

> ⚠️ **重要**: YES価格 + NO価格 ≈ $1.00（常に）

なぜ？1 YES + 1 NO = $1が保証されているからです。YES + NO < $1の場合、トレーダーは両方を購入して差額で利益を得ます。これが価格を均衡に保つ仕組みです。

## 価格変動

| イベント | 影響 |
|-------|--------|
| 結果に好材料 | YES価格 ↑、NO価格 ↓ |
| 結果に悪材料 | YES価格 ↓、NO価格 ↑ |
| 不確実性の増加 | 両方とも$0.50方向へ |
| 結果がほぼ確実 | YES → $0.95+、NO → $0.05 |
