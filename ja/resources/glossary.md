# 用語集

| 用語 | 定義 |
| ---- | ---------- |
| **AMM** | 自動マーケットメイカー — アルゴリズム取引プール（Uniswap v4） |
| **CLOB** | 中央指値注文板 — 従来型の注文マッチングエンジン |
| **Collateral（担保）** | アウトカムトークンの裏付けとしてデポジットされたUSDC |
| **Concentrated Liquidity** | 資本効率のために特定の価格範囲に集中したLPポジション |
| **Diamond Proxy** | EIP-2535アップグレードパターン — 単一プロキシ、複数の実装コントラクト（facets） |
| **Dynamic Fees（動的手数料）** | 満期が近づくにつれ0.5%から5%に増加するAMM手数料 |
| **ERC-20** | Ethereumトークン標準 — ファンジブル、DeFiとコンポーザブル |
| **ERC-2612** | 署名によるガスレストークン承認のPermit拡張 |
| **Facet** | Diamondプロキシ背後の実装コントラクト |
| **Hook** | プール操作にカスタムロジックを追加するUniswap v4プラグイン |
| **Merge（統合）** | 1 YES + 1 NOをバーンして1 USDCを受取 |
| **NOトークン** | イベントが発生しない場合に$1を支払うERC-20トークン |
| **Oracle** | イベントの結果を報告する外部データソース |
| **Outcome Token（アウトカムトークン）** | 予測市場でのポジションを表すYESまたはNOトークン |
| **Prediction Market（予測市場）** | 将来のイベントの結果について取引するマーケットプレイス |
| **Redeem（受取）** | マーケット解決後に勝利トークンをUSDCに交換 |
| **Slippage（スリッページ）** | 予想取引価格と実行取引価格の差 |
| **Smart Router** | CLOBとAMMの流動性を集約するPrediX Router |
| **Split（分割）** | 1 USDCをデポジットして1 YES + 1 NOトークンを受取 |
| **Spread（スプレッド）** | 最良買い価格と最良売り価格の差 |
| **TVL** | Total Value Locked — システムにデポジットされた総USDC |
| **USDC** | USD Coin — 担保として使用されるステーブルコイン（6 decimals） |
| **YESトークン** | イベントが発生した場合に$1を支払うERC-20トークン |
