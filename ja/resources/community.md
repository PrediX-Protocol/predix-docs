# コミュニティ

## 公式チャンネル

| プラットフォーム | リンク | 目的 |
| -------- | ---- | ------- |
| **Twitter / X** | [@CoinCrazeCen](https://x.com/CoinCrazeCen) | ニュース、アップデート、お知らせ |
| **Telegram** | [CoinCrazeCentral](https://t.me/CoinCrazeCentral) | コミュニティチャット、クイックアップデート |
| **GitHub** | [PrediX-Protocol](https://github.com/PrediX-Protocol) | ソースコード、イシュー、コントリビューション |

## 開発者向け

- **GitHub Issues**: バグ報告と機能リクエスト
- **ドキュメント**: 今読んでいるものです！

## トレーダー向け

- **Telegram**: マーケットのお知らせやディスカッションに参加
- **Twitter/X**: 最新のアップデートをフォロー

## コントリビューション

PrediXはコミュニティの貢献を歓迎します：

1. GitHubでリポジトリをFork
2. Featureブランチを作成
3. 明確な説明を添えてPull Requestを提出
