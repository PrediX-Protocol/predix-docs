# 予測市場

## 予測市場とは？

予測市場は、参加者が将来のイベントの結果について取引するマーケットプレイスです。アウトカムトークンの価格は、群衆の集合的な確率評価を反映します。

## 価格が確率を反映する仕組み

各マーケットは2つのトークンを作成します：

- **YESトークン**：イベントが発生すると$1.00を支払い
- **NOトークン**：イベントが発生しないと$1.00を支払い

YESが$0.70で取引されている場合、マーケットはイベント発生確率が**70%**であることを示唆します。

## 価格アンカー：YES + NO ≈ $1.00

誰でも以下が可能なため：

- 1 USDCを1 YES + 1 NOに**分割**
- 1 YES + 1 NOを1 USDCに**統合**

裁定取引者がこれを維持します：

```
YES価格 + NO価格 ≈ $1.00

YES + NO > $1.00 → 分割 + 両方売却 → 利益（価格下落）
YES + NO < $1.00 → 両方購入 + 統合 → 利益（価格上昇）
```

これはすべての予測市場の基本的な不変条件です。

## 歴史

| プラットフォーム | モデル | 状況 |
| -------- | ----- | ------ |
| Polymarket | Polygon上の中央集権型CLOB | 取引量で最大、ERC-1155トークン |
| Augur | Ethereum上の分散型AMM | 先駆者、限定的な採用 |
| Kalshi | 中央集権型取引所（CFTC規制） | 伝統的金融アプローチ |
| Omen (Gnosis) | Gnosis Chain上のAMMベース | コミュニティ主導 |
| **PrediX** | **Uniswap v4上のハイブリッドCLOB+AMM** | **ERC-20トークン、Diamondプロキシ** |
