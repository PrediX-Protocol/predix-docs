# ウォレット接続

## 必要なもの

- Web3ウォレット（MetaMask、Coinbase Wallet、Rainbowなど）
- ガス代用の少額のETH（Unichain上）

## ステップバイステップ

### ステップ1：PrediXを開く

ブラウザでPrediXトレーディングアプリを開きます。

### ステップ2：「Connect Wallet」をクリック

右上の**「Connect Wallet」**ボタンをクリックします。

### ステップ3：ウォレットを選択

リストからウォレットを選択：
- **MetaMask** — 最も人気のあるブラウザ拡張機能
- **Coinbase Wallet** — 初心者に簡単
- **WalletConnect** — モバイルウォレット用
- **Rainbow** — モバイルファーストウォレット

### ステップ4：接続を承認

ウォレットのポップアップが接続の承認を求めます。**「Connect」**または**「Approve」**をクリックします。

### ステップ5：Unichainに切り替え

プロンプトが表示されたら、**Unichain Sepolia**ネットワークに切り替えます：

```
Network Name: Unichain Sepolia
Chain ID: 1301
RPC URL: https://sepolia.unichain.org
Currency: ETH
```

> 💡 ほとんどのウォレットは接続時にネットワークを自動的に追加します。

### ステップ6：完了！

右上にウォレットアドレスが表示されます。接続完了です！

## ガス代用のETHが必要ですか？

Unichain上のトランザクションには少額のETHが必要です（取引あたり$0.01未満）。Unichain faucetからテストネットETHを取得してください。
