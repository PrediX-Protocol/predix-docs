# 初めての取引

YESシェアを購入してみましょう！約2分で完了します。

## ステップ1：マーケットを選ぶ

マーケット一覧を見て、興味のあるものを選びましょう：

```
「明日午後4時までにBTCは$85,000を超える？」
YES: $0.38    NO: $0.62
```

マーケットをクリックして取引パネルを開きます。

## ステップ2：YESまたはNOを選択

- 起こると思う → **YES**をクリック
- 起こらないと思う → **NO**をクリック

この例では**YES**を購入します。

## ステップ3：金額を入力

USDCをいくら使うか入力します。少額から：

```
金額: $25
```

見積もりが表示されます：

```
推定受取: ~65 YESシェア
1枚あたりの価格: ~$0.38
```

## ステップ4：「Buy YES」をクリック

緑色の**「Buy YES — $0.38」**ボタンをクリックします。

### 初めてですか？USDC承認

初回取引の場合、まずUSDCを**承認**する必要があります：

1. **「Approve USDC」**をクリック
2. ウォレットで確認
3. 数秒待つ
4. **「Buy YES」**をクリック

> 💡 承認は一度だけ必要です。以降の取引ではこのステップをスキップします。

## ステップ5：ウォレットで確認

ウォレットのポップアップが表示されます。詳細を確認して**「Confirm」**をクリック。

## ステップ6：完了！

YESシェアを所有しています！ポジションを確認：

```
あなたのポジション:
  YES  65シェア
```

## 次はどうなる？

| YESが勝った場合 | NOが勝った場合 |
|------------|-----------|
| 65シェア → $65 USDC | シェア → $0 |
| 利益: $65 - $25 = **$40** | 損失: **$25** |

**マーケット終了前にいつでも売却**して利益確定や損切りが可能です。

## 売却したい場合は？

1. **「Sell」**タブに切り替え
2. 売却するシェア数を入力
3. **「Sell YES」**をクリック
4. USDCがウォレットに返還

## まとめ

```
1. マーケット選択 → YESまたはNOを選ぶ
2. 金額入力 → $25
3. USDC承認（初回のみ）
4. 購入 → ウォレットで確認
5. 結果を待つか早期売却
```

おめでとうございます — あなたは予測市場トレーダーです！
