# テストUSDCの取得

PrediXはすべての取引に**USDC**（$1 USDにペッグされたステーブルコイン）を使用します。テストネットでは無料のテストUSDCを取得できます。

## Faucet — $10,000テストUSDCを取得

### ステップ1：接続確認

ウォレットが接続されていることを確認してください（[ウォレット接続](connect-wallet.md)参照）。

### ステップ2：「Faucet」または「Get USDC」をクリック

PrediXアプリで**Faucet**ボタンを見つけてクリックします。

### ステップ3：トランザクションを確認

ウォレットがトランザクションの確認を求めます。**「Confirm」**をクリックします。

> 💡 少額のガス代が必要です（ETHで$0.01未満）。

### ステップ4：USDCが到着

**$10,000テストUSDC**が数秒以内に残高に表示されます。取引の準備完了！

## 残高の確認

USDC残高は取引パネルの上部に表示されます：

```
残高: $10,000.00
```

## 重要な注意事項

> ⚠️ **テストUSDCには実際の価値はありません。** これはテストネットです — 学習とテスト専用。

> ⚠️ 必要に応じてfaucetから何度でもUSDCをリクエストできます。
