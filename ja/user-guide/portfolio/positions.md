# ポジションの確認

## 現在のポジション

ポートフォリオにシェアを保有するすべてのマーケットが表示されます：

```
ポジション

マーケット                           サイド   シェア    平均価格    価値        損益
"BTC $85K以上？"                    YES     65        $0.38       $24.70     -$0.38
"Chelsea vs Man City"               NO      100       $0.62       $62.00     +$5.00
"Trump 2028当選？"                  YES     200       $0.20       $40.00     +$12.00
```

## 各列の意味

| 列 | 意味 |
|--------|---------|
| **サイド** | YESまたはNO — どの結果に賭けたか |
| **シェア** | 保有トークン数 |
| **平均価格** | 1シェアあたりの平均購入価格 |
| **価値** | 現在価値 = シェア × 現在価格 |
| **損益** | 損益 = 現在価値 - コスト |
