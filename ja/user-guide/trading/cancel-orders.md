# 注文のキャンセル

## 指値注文のキャンセル

まだ約定していない指値注文がある場合：

1. **注文**セクションで未約定の注文を見つける
2. 注文の横の**「Cancel」**ボタンをクリック
3. ウォレットで確認
4. ロックされた資金が即座に返還

```
キャンセル前: $30 USDCがBUY注文にロック
キャンセル後: $30 USDCがウォレットに戻る
```

## 何が返還される？

| 注文タイプ | 返還されるもの |
|-----------|----------------|
| BUY YES/NO | USDC（デポジット） |
| SELL YES | YESトークン |
| SELL NO | NOトークン |

## マーケット満期後もキャンセルできる？

**はい！** マーケット満期後でもいつでも注文をキャンセルできます。資金が永久にロックされることはありません。

> 💡 マーケットが満期になっても注文が開いたままの場合、誰でもあなたの代わりにキャンセルできます — ただし返金は常に**注文者**に戻ります。

## 部分的に約定した注文

注文が部分的に約定した場合（例：100シェア中40シェア約定）：

```
元の注文: BUY 100 YES @ $0.35 → $35ロック
約定: 40シェア
残りキャンセル: 60シェア → $21 USDC返金
保有: 40 YESシェア
```
