# UIの理解

## メインレイアウト

取引ページは4つの主要エリアで構成されています：

```
┌──────────────────────┬─────────────────┐
│                      │ ポジション      │
│   価格チャート       │  YES: 65シェア  │
│                      ├─────────────────┤
│                      │  取引パネル     │
│                      │  購入 / 売却    │
│                      │  成行 / 指値    │
│                      │  [金額]         │
│                      │  [Buy YES]      │
├──────────────────────┤                 │
│   オーダーブック     │  分割 / 統合    │
│   買い    売り       │                 │
│   $0.35   $0.40      │  マーケット情報 │
│   $0.34   $0.41      │                 │
└──────────────────────┴─────────────────┘
```

## ポジション（右上）

このマーケットで現在保有しているものを表示：

```
ポジション
  YES    65シェア     ← YESトークン65枚保有
  NO     0シェア      ← NOトークン未保有
```

## 取引パネル（右）

### YES / NOボタン
```
┌─────────┐ ┌─────────┐
│  YES    │ │   NO    │
│  38¢    │ │   62¢   │
└─────────┘ └─────────┘
```
- 緑ハイライト = 選択中
- 表示価格 = 現在の市場価格

### 購入 / 売却トグル
- **購入**: USDCを使ってシェアを取得
- **売却**: シェアを売ってUSDCを取得

### 成行 / 指値トグル
- **成行**: 現在の価格で即座に売買
- **指値**: 自分の価格を設定して待機

### 金額入力
```
金額:                残高: $17,927.54
┌──────────────────────────────────────┐
│ $ 50.00                         MAX │
└──────────────────────────────────────┘
  [$25]  [$50]  [$75]  [$100]
```
- 金額を入力またはプリセットボタンをクリック
- **MAX** = 残高全額
- **残高**は参考表示

### 実行ボタン
```
┌──────────────────────────────────────┐
│         Buy YES — $0.38              │
└──────────────────────────────────────┘
```
実行する内容と価格を表示。

## 価格チャート（左）

時間経過による価格推移を表示：
- 緑の線 = 価格上昇
- 赤の線 = 価格下落
- ホバーで特定時点の価格を表示

## オーダーブック（左下）

```
価格    数量        合計
$0.40    61.58      543.73    ← アスク（売り注文）
$0.39    69.68      482.15
$0.38    70.34      412.47

         $0.38   スプレッド: $0.03      ← 中間価格

$0.35    24.67      24.67     ← ビッド（買い注文）
$0.34    31.87      56.54
$0.33    31.29      87.83
```

- **赤（上部）** = アスク — この価格で売りたい人
- **緑（下部）** = ビッド — この価格で買いたい人
- **スプレッド** = 最良買い価格と最良売り価格の差

## 分割 / 統合パネル

```
┌──────────────────────────────┐
│  [分割]          [統合]      │
│                              │
│  USDCを等量のYES + NO       │
│  トークンに分割              │
│                              │
│  ┌────────────────────┐      │
│  │ 100.00        USDC │      │
│  └────────────────────┘      │
│                              │
│  ┌────────────────────┐      │
│  │        分割         │      │
│  └────────────────────┘      │
└──────────────────────────────┘
```

- **分割**: $100 USDC → 100 YES + 100 NOトークン
- **統合**: 100 YES + 100 NO → $100 USDC
- 上級戦略に有用

## マーケット情報

```
マーケット情報
  カテゴリ:          crypto
  総担保:            $9,936.67 USDC
  ステータス:        active
  Oracle:            0x699a8c...992cf3
```

- **総担保**: このマーケットにロックされたUSDC総額
- **ステータス**: active / ended / resolved
- **Oracle**: 結果を決定する主体
