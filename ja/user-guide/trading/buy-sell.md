# シェアの売買

## シェアの購入（成行注文）

成行注文は現在の最良価格で即座に購入します。

### YESの購入
1. マーケット選択 → **YES**をクリック
2. **「Market」**注文タイプを選択
3. USDC金額を入力（例：$50）
4. 見積もり確認：「~130 YESシェア」
5. **「Buy YES — $0.38」**をクリック
6. ウォレットで確認 → 完了！

### NOの購入
同じ手順ですが**NO**をクリック：
1. **NO**をクリック
2. 金額を入力
3. **「Buy NO — $0.62」**をクリック

## シェアの売却

### YESの売却
1. **「Sell」**タブに切り替え
2. **YES**をクリック
3. シェア数を入力（または**MAX**をクリック）
4. 見積もり確認：「~$48 USDC」
5. **「Sell YES」**をクリック

### NOの売却
同様ですが**NO**を選択。

## 指値注文

特定の価格が欲しいですか？**指値注文**を使いましょう。

### 指値買い注文
1. **「Limit」**注文タイプに切り替え
2. 価格を設定（例：$0.30）
3. 数量を設定（例：100シェア）
4. **「Place Order」**をクリック
5. 誰かが$0.30で売るまで注文は待機

```
コスト: 100 × $0.30 = $30 USDC（約定またはキャンセルまでロック）
```

### 指値売り注文
1. **「Sell」**タブ → **「Limit」**
2. 価格を設定（例：$0.70）
3. 数量を設定
4. **「Place Order」**をクリック

### 指値注文のキャンセル
- 注文一覧で未約定の注文を見つける
- **「Cancel」**をクリック
- ロックされたUSDCまたはトークンが即座に返還

## 指値注文の価格ルール

```
最小価格: $0.01
最大価格: $0.99
刻み幅: $0.01（$0.35は可能ですが$0.355は不可）
最小注文: $1.00
最大未約定注文数: マーケットあたり50件
```

## 成行 vs 指値注文

| | 成行注文 | 指値注文 |
|---|---|---|
| スピード | 即座 | 指定価格まで待機 |
| 価格 | 現在の最良価格 | 指定した正確な価格 |
| 適している場合 | 素早い売買 | より良い価格の確保 |
| リスク | スリッページの可能性 | 約定しない可能性 |

> 💡 **ヒント**: スピード重視なら成行注文、価格重視なら指値注文を使いましょう。
