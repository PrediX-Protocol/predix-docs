# 分割と統合（上級）

> これは上級機能です。ほとんどのトレーダーには不要です。

## 分割（Split）とは？

**分割**はUSDCを等量のYESとNOトークンに変換します：

```
$100 USDCを分割 → 100 YESトークン + 100 NOトークン
```

## 統合（Merge）とは？

**統合**は逆の操作 — YESとNOを合わせてUSDCに変換：

```
100 YESトークン + 100 NOトークン → $100 USDC
```

## なぜ使う？

### 理由1：裁定取引

YES = $0.40、NO = $0.45（合計$0.85）の場合：

```
$1を分割 → 1 YES + 1 NO（コスト: $1）
YES $0.40売却 + NO $0.45売却 = $0.85
損失: $0.15 ← この方向では利益なし

ただしYES = $0.55、NO = $0.55（合計$1.10）の場合：
YES 1枚 ($0.55) + NO 1枚 ($0.55) = $1.10で購入
統合 → $1.00
損失: $0.10 ← これも利益なし

裁定取引は合計 ≠ $1.00の時のみ有効
```

### 理由2：両方のトークンを取得

マーケットメイカーとして両サイドを売りたい場合：

```
$500を分割 → 500 YES + 500 NO
SELL YES注文を$0.55で配置
SELL NO注文を$0.55で配置
両サイドから収益
```

### 理由3：ヘッジ

すでにYESを保有していてリスクを減らしたい場合：

```
100 YES保有（$0.40で購入）
$50を分割 → 50 YES + 50 NO
現在: 150 YES + 50 NO
NO勝利時: NOから$50受取（部分ヘッジ）
```

## 分割の方法

1. マーケットページに移動
2. **「Split / Merge」**セクションを見つける
3. **「Split」**タブをクリック
4. USDC金額を入力
5. **「Split」**をクリック
6. ウォレットで確認

## 統合の方法

1. **「Split / Merge」**セクション
2. **「Merge」**タブをクリック
3. 数量を入力（等量のYESとNOが必要）
4. **「Merge」**をクリック
5. USDCがウォレットに返還

> ⚠️ 統合には**等量**のYESとNOが必要です。YES 100枚、NO 50枚を保有している場合、50枚のみ統合可能です。
