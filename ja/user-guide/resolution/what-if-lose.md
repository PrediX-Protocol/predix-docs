# 負けた場合は？

## 簡単な回答

敗北したシェアは$0の価値になります。支払った金額だけを失います — それ以上はありません。

## 例

```
購入: YESシェア100枚 @ $0.35 → コスト: $35
マーケット解決: NO勝利
YESシェア: $0.00
損失: $35

$35以上を失うことは絶対にありません。
マイナス残高はありません。
マージンコールはありません。
```

## 損失を減らせる？

**はい — マーケット終了前に売却してください。**

```
YESを$0.35で購入
価格が$0.15に下落（YESに不利な状況）

選択肢A: 保有 → NO勝利時、1枚$0.35の損失
選択肢B: $0.15で売却 → 1枚$0.20の損失のみ（$0.15節約）
```

> 💡 **ヒント**: 予測に対する考えが変わったら、早めに売却して損失を限定しましょう。

## 最大損失

最大損失は**常に**支払った金額です：

| 購入価格 | 1枚あたりの最大損失 | 1枚あたりの最大利益 |
|-----------|-------------------|-------------------|
| $0.10 | $0.10 | $0.90 |
| $0.50 | $0.50 | $0.50 |
| $0.90 | $0.90 | $0.10 |

## 何かする必要がある？

**いいえ。** 敗北したシェアは単に価値がなくなります。ウォレットに残りますが$0の価値です。「クローズ」や「決済」する必要はありません。
