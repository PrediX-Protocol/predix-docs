# 賞金の受け取り（Redeem）

## マーケットが解決されたら

マーケットが終了し、オラクルが結果を決定した後：

```
マーケット: 「BTCは$85Kに到達する？」 → 結果: YES ✅

YESシェア → 1枚$1.00の価値
NOシェア → $0.00の価値
```

## 受け取り方法

1. 解決されたマーケットに移動
2. **「Redeem」**ボタンが表示されます
3. **「Redeem」**をクリック
4. ウォレットで確認
5. USDCが即座にウォレットに送金

```
保有: 65 YESシェア
結果: YES勝利
受取: 65 × $1.00 = $65 USDC → ウォレットへ
```

## 負けた場合は？

**敗北**シェアを保有している場合：

```
保有: 50 YESシェア
結果: NO勝利
YESシェア = $0.00
受け取るものなし。アクション不要。
```

> 追加の損失はありません — シェアの購入に支払った金額だけを失います。

## YESとNO両方を保有している場合は？

```
保有: 30 YES + 20 NO
結果: YES勝利

YES受取: 30 × $1.00 = $30 USDC ✅
NOシェア: 無価値 ($0) ❌
合計受取: $30
```

## 受取を忘れたら？

受取に**期限はありません**。解決後いつでも受け取れます。賞金は期限切れになりません。

## マルチアウトカムマーケット

「選挙の勝者は？」のようなマーケット：

```
保有:
  Trump YES: 50シェア
  Harris YES: 30シェア

勝者: Trump (index 0)

受取:
  Trump YES: 50 × $1.00 = $50 ✅
  Harris YES: 無価値 ❌

  また: Harris NOシェアが勝者！
  （Harrisが勝たなかったため）
```
