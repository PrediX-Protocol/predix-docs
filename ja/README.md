# PrediX Protocolへようこそ

PrediX Protocolは、**Uniswap v4**上に構築され、**Unichain L2**にデプロイされた完全分散型の予測市場プラットフォームです。

- **何を**: YES/NOトークンを使用して、現実世界のイベントの結果を取引します — 完全オンチェーン、トラストレス、DeFiとコンポーザブル。
- **なぜ**: ハイブリッドCLOB + AMM取引エンジンが最適な約定を実現。ERC-20アウトカムトークンがDeFiコンポーザビリティを解放。Diamondプロキシがシームレスなアップグレードを可能に。
- **どのように**: マーケットを作成し、アウトカムトークンを取引し、オラクルが解決し、賞金を受け取る — すべて数回のトランザクションで完了。
